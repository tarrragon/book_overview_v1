## Problem Analysis

Chrome 載入擴充功能後出現兩個錯誤：
1. Service worker registration failed. Status code: 15 — background.js 無法註冊為 Service Worker
2. Uncaught SyntaxError: Cannot use import statement outside a module — ES module import 語法在非 module context 中執行

Status code 15 = MANIFEST_ERROR_PAGE_IN_CONTENT_SCRIPTS_ONLY，通常表示 manifest 配置問題。
import 錯誤表示 background.js 使用了 ES module 語法但未在 manifest 中宣告 type: module。
[2026-03-28 14:03 更新] 根本原因：專案模組系統不一致。Logger.js 等核心模組使用 CommonJS (module.exports)，但 background.js 使用 ES module (import)。Chrome Extension Service Worker 以 type:module 載入時，無法解析 CJS export。需要統一模組系統或在 build 階段轉換。

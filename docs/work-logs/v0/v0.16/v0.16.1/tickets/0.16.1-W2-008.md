## Solution

<!-- To be filled by executing agent -->

---

建立 PreToolUse hook git-index-lock-cleanup-hook.py。偵測 git commit/add 前檢查 .git/index.lock，超過 10 秒自動移除（視為殘留），10 秒內輸出警告但不移除。註冊在 settings.json PreToolUse Bash 列表最前面。

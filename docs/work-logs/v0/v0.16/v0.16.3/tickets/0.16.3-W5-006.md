修正 git index.lock 殘留問題 — Hook 或流程異常中斷後未正確清理 lock 檔案

## Problem Analysis

根因：post-commit-fetch-hook.py 使用 Popen fire-and-forget 背景執行 git fetch，fetch 持有 index.lock 但 hook 已返回。主線程下一個 git 操作撞 lock。現有防護 git-index-lock-cleanup-hook.py（PreToolUse）閾值 10 秒不足。

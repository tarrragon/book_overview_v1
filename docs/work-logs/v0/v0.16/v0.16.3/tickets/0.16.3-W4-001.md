# Execution Log

## Task Summary

新增 stop hook 加入 session 經驗持久化提醒（清除前確認重要發現已記錄）

---

## Problem Analysis

### 問題根因

Session 中累積的決策經驗（流程改善洞察、架構判斷、除錯策略）不會自動記錄到文件。
Claude 在建立審查/分析 Ticket 後傾向建議 /clear，但這些 Ticket 的執行可能需要當前 session 的 context。

### 影響範圍

- Claude 的行為：建議 /clear 時未評估 context 損失風險
- 審查/分析類 Ticket 的品質：缺少建立時的討論背景

### 相關 Error Pattern

無直接相關。來源為 feedback memory `feedback_session_context_preservation.md`。

---

## Solution

建立 `session-experience-persistence-reminder-hook.py` 作為 Stop hook：

1. 跨版本掃描所有 Ticket，篩選今天建立、pending/in_progress、且包含審查/分析關鍵詞的 Ticket
2. 透過 stderr 輸出提醒訊息（Claude 讀取的 system context）
3. 提醒 Claude 評估 context 保存需求，禁止未評估就建議 /clear
4. 不阻止退出（`suppressOutput: true`），僅為提醒性質

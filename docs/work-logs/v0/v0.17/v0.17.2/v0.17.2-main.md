# v0.17.2 版本工作日誌

**版本號**: v0.17.2
**開始日期**: 2026-04-05
**目標**: 匯出/搜尋/Overview 適配 Tag-based Model
**狀態**: W1 規格完成，待 W2 實作

## 前情提要

v0.17.0 完成 Tag-based Book Model 核心重構（47 Ticket），v0.17.1 修復遺留測試（5 Ticket，0 failed / 4601 passed）。本版本完成 v0.17.x 里程碑剩餘範圍：匯出格式升級、搜尋引擎適配、Overview 頁面適配。

### 三大功能範圍

| 功能 | 現況 | 需要變更 |
|------|------|---------|
| Export (Interchange Format v2) | 多格式匯出（CSV/JSON），僅支援 v1 schema | 定義 v2 規格、tag 序列化、雙版本相容 |
| SearchEngine tag-based query | 純文字搜尋 | 新增 tag 篩選（AND/OR）、組合查詢 |
| Overview Page | 3 狀態顯示 | 6 狀態 + tag 欄位 + tag 篩選 UI |

### 已知技術債（v0.17.0 延後項，低優先）

1. validateField 純轉發零附加值 wrapper
2. 每個 CRUD 重複 loadTags/loadCategories IO
3. ID 生成模式散落 25+ 處

### 已知阻塞項

- Interchange Format v2 規格尚未定義（use-cases.md 標記「tag_tree 序列化格式待定義於匯出規格文件」）
- 需先完成規格再進入實作

## Solution

### 方向 1：Ticket → PM（snapshot 命令）

- 新建 `track_snapshot.py`：掃描所有版本 Ticket，產出格式化全局快照
- 整合到 `track.py`：import + dispatch（版本無關命令，同 `version` 特殊處理）
- 修正 import 路徑：`get_project_root` 從 `ticket_system.lib.paths`、`WORK_LOGS_DIR` 從 `constants`

### 方向 2：代理人 → Ticket（中間進度更新規範）

- 更新 `context-bundle-spec.md` v2.2.0：新增「代理人中間進度更新」章節
  - 定義四個更新時機：開始/階段完成/阻塞/完成
  - 更新標準 prompt 模板，包含進度更新指令
  - PM 行為表：查 Ticket 不查代理人

### PM SOP 整合

- 更新 `pm-role.md` v3.3.0：切換 SOP 從三步指令簡化為 `ticket track snapshot` 一條命令

## Problem Analysis

### Context Bundle

**規格來源**: docs/spec/overview-tag-display-spec.md (Section 4 + 5.2)

**測試目標**: 5 個測試案例驗證 Tag 篩選 widget 邏輯
1. tagFilterState 初始化為空 Set + OR 模式
2. setTagFilter 設定選中 tag 後 OR 篩選（書籍有任一 tag 即顯示）
3. setTagFilter AND 模式（書籍必須包含所有選中 tag）
4. clearTagFilter 清除篩選回到全部
5. 無 tag category 時 tagFilterState 仍可操作（graceful 降級）

**關鍵函式**:
- tagFilterState: { selectedTagIds: Set, mode: 'or'|'and' }
- setTagFilter(tagIds, mode) 或 addTagFilter/removeTagFilter
- filterByTags(books, selectedTagIds, mode) — 在 applyCurrentFilter 管線中
- clearTagFilter() — 清空 selectedTagIds

**現有 applyCurrentFilter** (L403-438): 目前只有 statusFilter -> textSearch -> sort
需擴充為 statusFilter -> tagFilter -> textSearch -> sort

**測試檔案**: tests/unit/overview/overview-page-controller.test.js
**實作檔案**: src/overview/overview-page-controller.js

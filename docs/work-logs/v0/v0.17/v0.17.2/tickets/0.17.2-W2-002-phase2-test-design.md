| 依賴 | Mock 策略 | 理由 |

## 測試檔案 1：format-version-detector.test.js

### 功能職責：偵測匯入資料的格式版本

---

### Group A：JSON v2 偵測（功能職責：v2 格式正確識別）

**A1：v2 明確 formatVersion 偵測 - 正常流程**

```
Given: JSON 資料含 metadata.formatVersion = "2.0.0"
Then（前置驗證）: data.metadata.formatVersion 存在且為字串
When: 呼叫 detectFormatVersion(data)
Then: 回傳 'v2'
```

測試資料：
```javascript
{ metadata: { formatVersion: '2.0.0' }, books: [] }
```

**A2：v2 formatVersion 以 "2." 開頭的其他版本**

```
Given: JSON 資料含 metadata.formatVersion = "2.1.0"
Then（前置驗證）: formatVersion 以 "2." 開頭
When: 呼叫 detectFormatVersion(data)
Then: 回傳 'v2'
```

測試資料：
```javascript
{ metadata: { formatVersion: '2.1.0' }, books: [] }
```

**A3：v2 結構特徵偵測（無明確版本號，有 readingStatus）**

```
Given: JSON 資料含 metadata 但無 formatVersion，books 中有 readingStatus 欄位
Then（前置驗證）: data.metadata 存在，data.books 為陣列
When: 呼叫 detectFormatVersion(data)
Then: 回傳 'v2'
```

測試資料：
```javascript
{ metadata: { exportDate: '2026-04-05' }, books: [{ id: '1', readingStatus: 'reading' }] }
```

---

### Group B：JSON v1 偵測（功能職責：v1 格式正確識別）

**B1：v1 純陣列格式**

```
Given: JSON 資料是純陣列（無 metadata 包裝）
Then（前置驗證）: Array.isArray(data) === true
When: 呼叫 detectFormatVersion(data)
Then: 回傳 'v1'
```

測試資料：
```javascript
[{ id: '1', title: '書', author: '作者', isNew: true }]
```

**B2：v1 含 books 但無 formatVersion**

```
Given: JSON 資料含 books 陣列但無 metadata.formatVersion
Then（前置驗證）: data.books 為陣列
When: 呼叫 detectFormatVersion(data)
Then: 回傳 'v1'
```

測試資料：
```javascript
{ books: [{ id: '1', title: '書', category: '小說' }] }
```

**B3：v1 含 metadata 但 books 無 readingStatus**

```
Given: JSON 資料含 metadata（無 formatVersion），books 中無 readingStatus
Then（前置驗證）: data.metadata 存在，books 不含 readingStatus
When: 呼叫 detectFormatVersion(data)
Then: 回傳 'v1'
```

測試資料：
```javascript
{ metadata: { exportDate: '2025-01-01' }, books: [{ id: '1', isFinished: true }] }
```

---

### Group C：無法辨識 / 邊界條件（功能職責：防禦性處理）

**C1：空物件**

```
Given: 輸入為空物件 {}
When: 呼叫 detectFormatVersion(data)
Then: 回傳 null
```

**C2：null 輸入**

```
Given: 輸入為 null
When: 呼叫 detectFormatVersion(data)
Then: 回傳 null
```

**C3：undefined 輸入**

```
Given: 輸入為 undefined
When: 呼叫 detectFormatVersion(data)
Then: 回傳 null
```

**C4：formatVersion 非 "2." 開頭**

```
Given: metadata.formatVersion = "1.0.0"，且 books 無 readingStatus
When: 呼叫 detectFormatVersion(data)
Then: 回傳 'v1'（降級為 v1 偵測，因有 books 陣列）
```

測試資料：
```javascript
{ metadata: { formatVersion: '1.0.0' }, books: [{ id: '1', category: '小說' }] }
```

**C5：books 為空陣列時的 v2 結構偵測**

```
Given: 有 metadata 但無 formatVersion，books 為空陣列
Then（前置驗證）: data.books.length === 0
When: 呼叫 detectFormatVersion(data)
Then: 回傳 'v1'（空 books 無法確認 readingStatus 特徵，降級為 v1）
```

測試資料：
```javascript
{ metadata: { exportDate: '2026-01-01' }, books: [] }
```

---

### Group D：CSV 版本偵測（功能職責：CSV header 識別）

**D1：CSV 含 readingStatus 欄位**

```
Given: CSV headers 陣列含 'readingStatus'
When: 呼叫 detectCsvVersion(headers)
Then: 回傳 'v2'
```

測試資料：`['id', 'title', 'readingStatus', 'authors', 'progress']`

**D2：CSV 含 isNew/isFinished 欄位**

```
Given: CSV headers 陣列含 'isNew' 或 'isFinished'
When: 呼叫 detectCsvVersion(headers)
Then: 回傳 'v1'
```

測試資料：`['id', 'title', 'author', 'isNew', 'isFinished', 'progress']`

**D3：CSV 兩者皆無 - 降級 v1**

```
Given: CSV headers 不含 readingStatus 也不含 isNew/isFinished
When: 呼叫 detectCsvVersion(headers)
Then: 回傳 'v1'
```

測試資料：`['id', 'title', 'author', 'category']`

---

### Group D 跨群組依賴：無。可獨立執行。

## 測試檔案 2：v1-to-v2-converter.test.js

### 功能職責：v1 格式書籍轉換為 v2 格式

---

### Group E：convertV1ToV2Book 欄位轉換（功能職責：單本書轉換）

**E1：author 字串轉 authors 陣列 - 正常流程**

```
Given: v1Book 含 author = "劉慈欣"
Then（前置驗證）: typeof v1Book.author === 'string'
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 authors 為 ["劉慈欣"]
```

**E2：author 為空字串時轉為空陣列**

```
Given: v1Book 含 author = ""
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 authors 為 []
```

**E3：author 為 undefined 時轉為空陣列**

```
Given: v1Book 不含 author 欄位
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 authors 為 []
```

**E4：isFinished=true 轉為 readingStatus='finished'**

```
Given: v1Book 含 isFinished = true, progress = 50
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 readingStatus 為 'finished'（isFinished 優先）
Then: 結果不含 isNew 和 isFinished 欄位（已移除）
```

**E5：progress > 0 且 isFinished=false 轉為 'reading'**

```
Given: v1Book 含 isFinished = false, progress = 50
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 readingStatus 為 'reading'
```

**E6：progress = 0 且無 isFinished 轉為 'unread'**

```
Given: v1Book 含 progress = 0, 無 isFinished 欄位
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 readingStatus 為 'unread'
```

**E7：異常組合 isNew=true + isFinished=true**

```
Given: v1Book 含 isNew = true, isFinished = true
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 readingStatus 為 'finished'（isFinished 優先）
```

**E8：progress 正規化（字串 "75" 轉為數字 75）**

```
Given: v1Book 含 progress = "75"
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 progress 為 75（數字）
Then: 結果的 readingStatus 為 'reading'
```

**E9：progress 超出範圍 clamp（150 -> 100）**

```
Given: v1Book 含 progress = 150
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 progress 為 100（clamp 上限）
Then: 結果的 readingStatus 為 'finished'
```

**E10：progress 為負數 clamp（-10 -> 0）**

```
Given: v1Book 含 progress = -10
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果的 progress 為 0（clamp 下限）
Then: 結果的 readingStatus 為 'unread'
```

**E11：新增預設欄位驗證**

```
Given: v1Book 為最小 v1 書籍（只有 id, title）
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果含 tagIds = []
Then: 結果含 isManualStatus = false
Then: 結果含 source = 'readmoo'
Then: 結果含 updatedAt（ISO 8601 格式字串）
```

**E12：category 欄位保留為 tagNames（轉換後處理）**

```
Given: v1Book 含 category = "科幻小說"
When: 呼叫 convertV1ToV2Book(v1Book)
Then: 結果不含 category 欄位（已移除或轉移）
```

---

### Group F：convertV1CategoryToTag 分類轉換（功能職責：category -> tag 結構）

**F1：正常 category 陣列轉換**

```
Given: categories = ["科幻", "推理"]
When: 呼叫 convertV1CategoryToTag(categories)
Then: 回傳結構含一個 TagCategory，name = "匯入分類"
Then: 回傳結構含兩個 Tag，name 分別為 "科幻" 和 "推理"
Then: 每個 Tag 的 categoryId 指向 "匯入分類" 的 id
```

**F2：空字串不建立 tag**

```
Given: categories = ["科幻", "", "推理"]
Then（前置驗證）: categories 含空字串元素
When: 呼叫 convertV1CategoryToTag(categories)
Then: 回傳結構只含 2 個 Tag（"科幻"、"推理"），空字串被過濾
```

**F3："未分類" 不建立 tag**

```
Given: categories = ["未分類", "科幻"]
When: 呼叫 convertV1CategoryToTag(categories)
Then: 回傳結構只含 1 個 Tag（"科幻"），"未分類" 被過濾
```

**F4：空陣列輸入**

```
Given: categories = []
When: 呼叫 convertV1CategoryToTag(categories)
Then: 回傳結構含 TagCategory "匯入分類"
Then: tags 為空陣列
```

**F5：重複 category 值去重**

```
Given: categories = ["科幻", "科幻", "推理"]
When: 呼叫 convertV1CategoryToTag(categories)
Then: 回傳結構只含 2 個 Tag（"科幻" 出現一次、"推理"）
```

**F6：全部為空字串和 "未分類"**

```
Given: categories = ["", "未分類", ""]
When: 呼叫 convertV1CategoryToTag(categories)
Then: 回傳結構含 TagCategory "匯入分類"
Then: tags 為空陣列（全部被過濾）
```

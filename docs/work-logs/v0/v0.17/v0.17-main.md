## 背景

**版本目標變更**（2026-04-03）：
- 原目標：測試修復與品質恢復（失敗測試歸零）
- 新目標：實施 PROP-007 核心 model 變更
- 原因：tag-based model 變更影響所有測試邏輯，先重構 model 再重寫測試
- 原有 Ticket 已遷移至 v0.18.0（測試重寫版本）

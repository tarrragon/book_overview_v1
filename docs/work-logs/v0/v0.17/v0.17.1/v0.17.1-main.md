# v0.17.1 版本工作日誌

**版本號**: v0.17.1
**開始日期**: 2026-04-05
**目標**: 修復 v0.17.0 遺留的 26 個失敗測試 + 技術債清理
**狀態**: 已發布（tag v0.17.1）

## 前情提要

v0.17.0 完成 Tag-based Book Model 重構（PROP-007），47 Ticket 全部完成，4575 tests passed。但遺留 3 個失敗 test suite（26 tests），屬預存問題非 v0.17.0 引入。本版本目標為修復這些失敗測試，確保測試通過率回到 100%。

### 失敗測試摘要

| Suite | 失敗數 | 根因摘要 |
|-------|--------|---------|
| duplicate-handling-v2.test.js | ~25 | 跨平台去重邏輯 — 不同 ID 的同書（title+author）未被識別合併 |
| data-integrity-recovery.test.js | ~1 | `performIntegrityCheck is not a function` — 函式引用斷裂 |
| popup-extraction-service.test.js | ~1 | timestamp 精度 1ms 差異（flaky test） |

### 已知技術債（v0.17.0 多視角審查延後項）

1. validateField 純轉發零附加值 wrapper（低優先）
2. 每個 CRUD 重複 loadTags/loadCategories IO（效能優化）
3. ID 生成模式散落 25+ 處（分析結論暫不統一）

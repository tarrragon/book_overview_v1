## Problem Analysis

### 根因（WRAP 分析）

**Bug 位置**：
1. `acceptance_checkers/ticket_parser.py:32` — `extract_children_from_frontmatter`
2. `acceptance_checkers/ana_spawned_checker.py:31` — `extract_spawned_tickets_from_frontmatter`

**問題**：兩個函式都假設 frontmatter 欄位（children/spawned_tickets）是字串格式，呼叫 `.strip()`。但 YAML 解析後這些欄位回傳 Python `list`，導致 `AttributeError: 'list' object has no attribute 'strip'`。

**影響**：
- 異常被 acceptance-gate-hook 的 try-except 捕獲
- Hook 回傳 allow（exit 0），children/spawned 檢查被完全繞過
- 所有有 children 或 spawned_tickets 的 Ticket 的 complete 檢查都失效

**時間線確認**：
- W3-001 拆分 acceptance-gate-hook 時引入此 bug（拆分過程中型別假設錯誤）
- W4-001 是拆分後第一個有 children 的 ANA Ticket → 首次觸發但靜默失敗

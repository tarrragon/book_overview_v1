## 修復內容

### 問題確認
AST 分析發現 2 個 Hook 在同一執行路徑輸出 2 個 JSON 到 stdout：
1. bash-edit-guard-hook.py: _print_warning_message() 輸出 additionalContext JSON，然後 main() 再輸出 permissionDecision JSON
2. pre-test-hook.py: 有 issues 時先輸出 additionalContext JSON，再輸出 permissionDecision JSON

### 修復方式
合併為單一 JSON 輸出，將 additionalContext 和 permissionDecision 放在同一個 hookSpecificOutput 中。

### 影響範圍
- bash-edit-guard-hook.py: _print_warning_message() 改為 return 字串，main() 合併輸出
- pre-test-hook.py: 移除條件式第一個 print，合併到最終 result

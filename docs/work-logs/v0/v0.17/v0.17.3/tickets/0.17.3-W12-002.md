### 修改內容

1. 新增 dispatch-record-hook.py（PreToolUse:Agent）— 派發時記錄到 dispatch-active.json
2. 修改 active-dispatch-tracker-hook.py（PostToolUse:Agent）— 清理後自動報告剩餘活躍派發數量
   - [WAIT] 仍有 N 個代理人在執行: Agent A, Agent B
   - [OK] 所有代理人已完成，可開始驗收
3. 註冊 dispatch-record-hook 到 settings.json PreToolUse:Agent

完整生命週期：record（派發前）→ clear + report（完成後）

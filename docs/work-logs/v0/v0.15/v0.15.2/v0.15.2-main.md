# v0.15.2 版本工作日誌

**版本號**: v0.15.2

## 前情提要

v0.15.1 修復建置問題後，實際載入 Extension 時發現 Chrome runtime 錯誤：MESSAGE_EVENTS、CONNECTION_EVENTS 等事件常數在 background service worker 中未定義。這些常數在 v0.15.0 的 ErrorCodes 分層重構中被移動但引用未更新。

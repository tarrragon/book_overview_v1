# v0.15.4 版本工作日誌

**版本號**: v0.15.4
**開始日期**: 2026-03-28
**目標**: Overview 頁面修復 + 提取流程容錯強化
**狀態**: 已完成

---

## 前情提要

v0.15.3 修復 Popup UI 後，使用者實際操作發現 Overview 頁面完全無法載入（event-bus.js require 錯誤、ErrorCodes 重複宣告、chrome-event-bridge.js 找不到）。同時提取流程的日誌不足，排查問題困難。

---

## 開發日誌

### 2026-03-28：W1 — Overview 頁面緊急修復（5 個 Ticket）

- W1-001 修復 event-bus.js `require is not defined`（CommonJS/ESM 不相容）

- W1-003 修復 chrome-event-bridge.js ERR_FILE_NOT_FOUND
- W1-004 修復封面 URL 安全過濾警告大量洗版 console
- W1-005 推進版本號至 0.15.4

### 2026-03-28：W2 — 提取流程日誌補齊

- W2-001 補齊 book-data-extractor 提取結果為空時的日誌
- W2-002 補齊 parseBookElement 解析失敗時的逐筆日誌
- W2-003 修正 extractAllBooks 全部解析失敗時日誌等級從 warn 升為 error

### 2026-03-28：W3 — 選擇器修復與容錯重構

- W3-003 修復 Service Worker error handler 延遲註冊警告
- W3-004 重構 parseBookElement 採用容錯策略（必要/可選欄位分離）

### 2026-03-28：W4 — 文件與測試

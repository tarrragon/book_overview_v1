# v0.20.0 — Tag-based Book Model 重構（Chrome Extension 側）

## 版本目標

配合 Flutter App v0.32.0 PROP-007，將 Chrome Extension 的 Book Model 改為 tag-based metadata 架構。

## 對應 Flutter App 版本

| Extension 版本 | App 版本 | 內容 |
|---------------|---------|------|
| v0.20.0 | v0.32.0 | Tag-based model + Interchange Format v2 |

## Wave 規劃

### W1: Model + Storage 重構

- 閱讀狀態擴充（3 種 → 6 種：新增 queued, abandoned, reference）
- Chrome Storage 資料結構適配 tag-based model
- tag_categories + tags 資料結構（Chrome Storage 版本）

### W2: 匯出重寫

- 匯出為 Interchange Format v2（含 tags 按類別 + tag_tree）
- 向下相容舊格式匯出選項

### W3: 搜尋和顯示適配

- SearchEngine 適配 tag-based 查詢
- Overview Page 顯示 tag-based 資訊

### W4: 中文圖書分類法 + Tag 管理

- 預裝中文圖書分類法 tag 樹
- Tag 管理功能（新增/重命名/刪除）

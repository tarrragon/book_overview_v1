### 場景組 C：cascade 分層刪除（tag-storage-adapter.js）

deleteTagCategory 重寫為樹狀分層語意，新增 options.cascadeSubtree（預設 false）：
- C4 isSystem（含 Uncategorized）→ error `system_protected`（既有 `cannot_delete_system` 改名，TD-1；grep 確認無測試依賴舊字面）
- C1/C3-noopt 非葉且未 opt-in → error `has_children` + hint（HAS_CHILDREN_HINT 常數）
- C2 葉節點 → ensureUncategorizedCategory（惰性建立確定性 ID `sys_cat_uncategorized`，isSystem=true）+ tag.categoryId 轉移、tag 不刪、Book.tagIds 不變；寫入序 TAGS→CATEGORIES
- C3 cascadeSubtree=true → collectSubtreeIds 後序走訪（葉到根）刪整子樹 + 其下 tag + 清 Book.tagIds 引用
- C-rollback：自管 try/catch 原子回滾（既有 withAtomicRollback 無 cascade_partial 語意且 rollback 寫入再失敗會拋出），失敗回 `{success:false, error:'cascade_partial', rolledBack:true}`，rollback 寫入二次失敗亦吞掉保證契約回傳

新增輔助：ensureUncategorizedCategory / collectSubtreeIds；錯誤碼集中於 TAG_CATEGORY_OPERATION_ERROR_CODES（Object.freeze，與 TagSchema 驗證類錯誤碼分離）。

### 場景組 D：預裝載入（tag-storage-adapter.js + background.js + 新資料檔）

- 新增 `src/data-management/presets/chinese-classification.js`：賴永祥分類法精簡集（10 主類 + 6 代表性次類，確定性 ID `sys_cat_*`）+ UNCATEGORIZED_CATEGORY_ID/NAME 常數。完整 110 節點待 W2-003 授權（TD-6），僅替換本檔不動載入機制。

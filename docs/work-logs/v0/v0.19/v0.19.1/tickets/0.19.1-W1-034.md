### 變更摘要

調查發現 template serena 區段（settings-local-template.json）比預期更亂：混用 `mcp__serena__` 與 `mcp__plugin_serena_serena__` 兩種前綴，且 list_dir / get_symbols_overview 在兩種前綴下重複。

### 修正
- 6 處 `mcp__plugin_serena_serena__*` 全部統一為 `mcp__serena__*`（對齊 CLAUDE.md user-level 設定 + session 實機暴露前綴）
- 去除轉換後產生的重複（list_dir、get_symbols_overview）
- 新增 `_COMMENT_SERENA` 條件說明註解：前綴依安裝方式而定（user-level=mcp__serena__ / plugin marketplace=mcp__plugin_<name>_serena__），新專案應依實機 ToolSearch 暴露名核對

### 驗證
- `python3 -c "import json; json.load(...)"`：valid JSON
- `rg plugin_serena_serena .claude/templates/`：0 殘留

serena tool API 本身漂移：template 與 settings.local.json 仍引用 7 個不在本 session serena 實機暴露清單的工具名（list_dir/find_file/search_for_pattern/activate_project/read_file/create_text_file/check_onboarding_performed）。屬權限 allowlist dead entries（低危害：不 match 即不生效，不破壞功能）。已在 template 加註「依實機核對」緩解。本 session 累計在 3 檔發現 MCP 工具名漂移（messages.py / search-tools-guide / settings template），疑為系統性，建議評估是否做一次全 .claude/ MCP 工具名 audit。

### Token 設計

新增 `src/core/design-system/shadows.js`，匯出 `SHADOW_COLORS`（sm=rgba(0,0,0,0.1) 輕陰影、
md=rgba(0,0,0,0.25) 中陰影）與 `OVERLAY_COLORS`（scrim=rgba(0,0,0,0.5) modal backdrop）。
產生器 data-driven 產出 `--shadow-color-sm` / `--shadow-color-md` / `--overlay-scrim`。

僅收斂「顏色」維度而非完整 box-shadow：5 處 box-shadow 的 offset/blur 各異
（0 0 10px / 0 2px 8px / 0 2px 4px / 0 2px 10px / 0 4px 20px），故 CSS 寫法為
`box-shadow: <offset/blur> var(--shadow-color-*)`，offset/blur 仍由各規則宣告，
避免強行統一不同元件層級。overlay 為完整背景值 `var(--overlay-scrim)`。

### 收斂位置（8 處）

overview.css L36/152/210 box-shadow → --shadow-color-sm；L433 overlay → --overlay-scrim；
L446 modal box-shadow → --shadow-color-md；檔頭註解移除「例外...保留」字樣改述已收斂。
popup.html L23/48 box-shadow → --shadow-color-sm。

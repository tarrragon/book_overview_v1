### 實作摘要（落地內容）

1. **colors.js**：STATUS_COLORS 改 D3 深字淺底 6 組（unread #546E7A/#ECEFF1 等，全達 WCAG AA >=4.68:1）；移除 GRADIENT 常數與 hexToRgba 函式。
2. **index.js**：同步移除 GRADIENT 的 destructure 與 re-export（唯一 GRADIENT 消費點，全專案 grep 確認無外部引用）。
3. **scripts/generate-design-system-css.js（新建）**：require 3 個 design-system JS 模組，data-driven 產出 :root 變數（camelCase 轉 kebab-case），檔頭含 AUTO-GENERATED DO NOT EDIT。
4. **scripts/build.js**：頂層 require 產生器；build() 開頭（複製 src/ 前）呼叫 generate()，generated CSS 隨 filesToCopy 的 src/ 一併複製進 build 目錄。
5. **package.json**：新增 validate:design-system（產生器 + git diff --exit-code）。
6. **tests/unit/design-system-css-snapshot.test.js（新建）**：16 個斷言驗證關鍵 token（--color-primary、--status-unread-fg 等）+ 格式 + STATUS_COLORS 單一來源無漂移。

### 設計決策

- design-system.css 含 `--color-dark-background`（colors.js 的 darkBackground key），W2-001 :root 範例未列但屬合法 source token；data-driven 產生器迭代全部 COLORS key 為正確 SSOT 行為。

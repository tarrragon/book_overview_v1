## Solution

### 決策：採方案 B（移除誤導設定 + 文件記錄 install_extension 絕對路徑為標準步驟）

**為何不採方案 A（絕對路徑寫入 .mcp.json）**：
- 絕對路徑會把開發者本機路徑（如 `/Users/tarragon/Projects/book_overview_v1/build/development`）寫進 tracked config，跨機器不可移植
- 其他開發者 clone repo 後該設定立即失效，反而比現狀更具誤導性
- 相對路徑失效已由 source ticket W1-001.1 實證（line 174），絕對路徑能載入但維護成本反向上升

**為何方案 B 是正解**：
- W1-001.1 已實證 `install_extension(path="<絕對路徑>")` 動態載入是唯一可靠路徑
- `.mcp.json` 只保留模式類旗標（`--isolated` + `--categoryExtensions`），不再含本機路徑
- 文件層在 SKILL Workflow A + readmoo.md Step 1 雙處明示絕對路徑必要性

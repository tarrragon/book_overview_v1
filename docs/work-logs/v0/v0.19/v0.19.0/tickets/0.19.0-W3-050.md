## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->
### 變更摘要

**檔案**：`.mcp.json`（codegraph 區塊）

**變更內容**：
- `command`: `codegraph` → `codegraph-mcp`
- `args`: `[serve, --mcp, --path, .]` → `[-w, .]`

**根因分析**：
1. W6-001.1 文件「npm install -g codegraph」實際安裝 npm registry 上的 codegraph@1.0.0（469B placeholder，無 bin 入口），非實際 codegraph CLI
2. 真正 codegraph MCP server 為 `@astudioplus/codegraph-mcp@0.16.6`（maintainer: anvanster-personal，bin: `codegraph-mcp`）
3. 新版 codegraph-mcp 預設啟用 MCP 模式，`--mcp` flag 變多餘且重複指定會報 `cannot be used multiple times`
4. 子命令格式從 `serve --mcp --path .` 簡化為 `-w .`

**安裝指令更新**（W6-001.1 development-setup.md 後續修正待建 ticket）：
```bash
# 舊（無效）
npm install -g codegraph

# 新（正確）
npm install -g @astudioplus/codegraph-mcp
```

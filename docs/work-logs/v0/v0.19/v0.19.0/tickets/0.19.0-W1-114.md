- **症狀**：修復 ticket 測試綠燈但 runtime 失效；後續 ticket 基於「修復已生效」假設推進造成連鎖回歸
- **根因**：mock 替代真實依賴 + 斷言不檢查訊息文字 + JS 靜默忽略多餘參數三層共振

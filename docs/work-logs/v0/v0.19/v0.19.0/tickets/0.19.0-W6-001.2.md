### 關鍵設計決策

1. **codegraph binary 偵測順序**：codegraph-mcp（0.16.6+ 主名稱）→ codegraph（舊版 fallback）。處理 W3-050 揭露的 npm scoped name 遷移
2. **codegraph 版本參數**：用 `--info` 而非 `--version`（0.16.6 不支援 --version）
3. **cbm 索引狀態**：固定 INDEX_MCP_MANAGED（CLI 不暴露 index_status）
4. **codegraph 索引狀態**：以 `.codegraph/` 目錄存在性判定 INDEX_OK / INDEX_MISSING；STALE 暫不實作（codegraph-mcp 內建 file watcher debounce ~500ms，STALE 通常為瞬時狀態）
5. **整合測試在 source venv 跑**：CLI entry point (`/Users/tarragon/.local/bin/project-init`) 缺 `pyproject_scanner` env dep（預先存在問題，非 W6-001.2 引入），acceptance #2 邏輯層滿足但 CLI 整合驗證需 follow-up（見 NeedsContext + Completion Info）

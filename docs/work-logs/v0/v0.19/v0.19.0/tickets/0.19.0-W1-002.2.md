### 任務範圍說明

本 ticket 為 ZIP 安裝實機驗證，原規劃 3 個 Chrome profile (A/B/C) 經 0.19.0-W1-064 ANA 評估後採方案 B 精簡為 1 clean profile 驗證 + 自動化前置檢查補充。

### 減裁依據（引用 0.19.0-W1-064 結論）

| 原條目 | 決策 | 理由 |
|---|---|---|
| Profile B (multi-ext) | 移除 | permissions 僅 storage + activeTab，衝突風險極低（W1-064 視角 3 分析） |
| Profile C (dev mode off) | 移除 | 測試的是 Chrome 原生行為，非 extension 功能 |
| 安裝時間 < 30 秒 | 移除 | Load Unpacked 是檔案系統操作，秒級完成，無鑑別力 |
| 截圖記錄 | 降級為選填 | 內測階段文字描述即足夠 |
| 自動化前置檢查 | 拆出為 0.19.0-W1-066 | manifest + SW + CS + permissions 由 agent 自動驗證 |
| 用戶手動 checklist | 拆出為 0.19.0-W1-067 | 用戶操作 4 步驟指引另立 ticket |

## Solution
<!-- Schema[DOC/Solution]: 免填（DOC 類型以變更摘要取代）（.claude/pm-rules/ticket-body-schema.md） -->
### 變更摘要

1. `README.md`：新增「權限與隱私聲明」章節（置於「授權條款」之前），含：
   - 單一用途說明
   - 權限使用理由表（`storage` / `activeTab` / `*://*.readmoo.com/*` host permission），並聲明不使用 tabs / `<all_urls>` / webRequest / 遠端程式碼
   - 資料用途聲明表（蒐集資料 / 儲存位置 / 資料傳輸 / 個人識別資訊 / 分析與廣告 / 匯出檔案）
   - 隱私權政策小節
2. 新增根目錄 `PRIVACY.md`：完整隱私權政策（概述 / 蒐集的資料 / 資料使用方式 / 資料儲存與傳輸 / 第三方服務 / 使用者控制權 / 權限說明 / 政策變更 / 聯絡方式）。
3. `README.md` 隱私權政策小節改為連結 `PRIVACY.md`（無 broken link）。

### 設計決策

權限內容依當下 `manifest.json` 實際 permission 撰寫（storage / activeTab / host_permissions）。送審時的權限變更同步由 W5-006 正規化的 version-release「權限需求變更檢查」流程涵蓋。

### 待處理（送審前）

README 與 PRIVACY.md 內的 GitHub URL 仍為佔位符 `your-username/multi-bookstore-manager`，Chrome Web Store 隱私權政策 URL 欄位需可公開託管的網址——送審前須換成真實 repo 網址。

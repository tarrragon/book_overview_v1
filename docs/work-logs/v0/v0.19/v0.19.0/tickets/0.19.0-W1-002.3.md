- 開啟後左上角出現「載入未封裝項目」/「封裝擴充功能」/「更新」三按鈕

##### 章節 4：Load Unpacked 載入 Extension

- 點「載入未封裝項目」（Load unpacked）
- 選擇章節 2 解壓後的根目錄（含 manifest.json 那層）
- 卡片出現於頁面，顯示名稱「Readmoo 書庫數據提取器」+ 版本號 + Service worker 連結
- 銜接第 5 章「安裝後驗證 checklist」做完整驗證

##### FAQ（>= 3 題）

Q1: 開發者模式 toggle 找不到或灰色
- 可能原因：Chrome 企業管控、版本過舊、特殊使用者群組
- 處理：確認 Chrome 版本 >= 88，使用個人 profile，企業環境可詢問 IT

Q2: Service Worker DevTools 紅色 error
- 可能原因：manifest 結構問題、host_permissions 衝突、依賴載入失敗
- 處理：截圖 error message + stack trace 回報內測負責人；先試卡片 reload 按鈕；確認預先執行 `npm run validate:manifest:prod` 已通過（W1-066）

Q3: Readmoo 頁面 content script 未注入（popup 顯示「不支援當前頁面」）
- 可能原因：URL 未進入 library hash、未登入、其他 extension 衝突
- 處理：確認 URL `https://read.readmoo.com/#/library`、完成 Readmoo 登入、暫時停用其他 extension 排查

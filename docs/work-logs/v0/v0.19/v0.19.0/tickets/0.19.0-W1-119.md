#### TD-2（**P2 / 中優先 / 雙視角共識**）：MessageDictionary → Logger 循環依賴

**spawn**：0.19.0-W1-121 IMP P2

#### TD-3（**P2 / 中優先 / cinnamon 單視角**）：Logger.js fallback 影子 messages dict

**來源**：cinnamon TD-2

**問題**：`Logger.js:25-35` 條件性 require fallback 定義迷你 dict（DEBUG/INFO/WARN/ERROR_MESSAGE 4 keys），與 _loadDefaultMessages 24 keys 不一致，等同「影子 GlobalMessages」。Chrome Extension bundled 後實際走不到（dead code 嫌疑）。

**spawn**：0.19.0-W1-122 IMP P2

1. **訊息系統作用域顯性化**：從「單一 GlobalMessages 全部混合」演進為「GlobalMessages（~24 跨模組共用） + N 個 module-local dict（extractor/popup/search-filter/validator/background）」

3. **依賴注入升級**：Logger 接受 messages 參數，可使用模組 local dict

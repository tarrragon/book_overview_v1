### 設計決策

**UC01ErrorAdapter / UC03ErrorAdapter / UC05ErrorAdapter / UC06ErrorAdapter** 映射表快取測試從「duration < 10ms」改為「toBe(firstMapping) 參考相等」是更強的快取驗證（沿用 W1-017 UC07 toEqual→toBe 同一原則）。原計時斷言即使環境波動通過也只證明「快」，無法證明「真的用了快取」；參考相等斷言能直接驗證 Object.freeze 單例未重建。

**所有「duration < N」純計時測試**重寫為「支援批量呼叫而不崩潰」是 W1-094 同一保留功能 + 移除計時的折衷模式：保留 loop 證明不崩潰，新增 length 與邊界存在斷言取代計時當作 functional 簽名。

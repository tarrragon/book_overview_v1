| `get_active_version`（單數） | `handoff-auto-resume-stop-hook.py:152` | 檔內無呼叫、跨模組無 import — 純 dead code |
| `get_active_versions`（**複數**，回傳 `list[str]`） | `ticket_system/lib/version.py:39` | track_query.py / track_stale_list.py / track_stuck_anas.py import 的是**這個** |

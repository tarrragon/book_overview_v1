
### 影響範圍

- 主要影響檔：tests/performance/baseline-performance.test.js
- 4 處潛在敏感點：line 88-89 (popupOpen) / line 215-216 (smallBookExtract) / line 251-252 (mediumBookExtract) / line 290-291 (JSON 解析動態 actualSize × 2)
- 同檔另含計時斷言（line 82/124/132/168/178/179/209/245/278/371）需評估是否同問題

### 根因類別

**環境敏感記憶體斷言**（test-assertion-design-rules.md 規則 1 變體）：
- jsdom 環境 + jest workers 並行 → GC 觸發時機不確定
- `process.memoryUsage().heapUsed` 量測值受 V8 內部 GC、JIT 編譯、其他並行 test 的記憶體壓力影響
- 規則 1 主要針對計時斷言，記憶體斷言為同類但更敏感（GC pause 隨機性 > CPU 排程隨機性）

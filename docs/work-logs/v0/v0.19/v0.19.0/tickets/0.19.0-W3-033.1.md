### 問題根因

`references/plugin-management.md`（v1.0.0, 2026-04-20）建立時 CC 尚未引入 dependency enforcement 與 projected context cost 等機制，導致規範與 CC v2.1.143 實際平台行為脫節。具體缺口：第 1 章評估清單缺依賴性檢查項、Token 評估仍靠人工估算、卸載流程未說明 dependency 限制、LSP 查詢無 CLI 指令、反模式缺根目錄 SKILL.md 曝光警告。

### 影響範圍

- `.claude/references/plugin-management.md`：第 1/2/4/5/7 章均需更新

### 相關 Error Pattern

W3-033 ANA 結論：PC-088（工具選擇偏誤）間接相關，projected context cost 可補強評估清單第 4 題的可執行性。

產出檔：.claude/skills/compositional-writing/references/dry-run-guide.md

設計了 3 個測試場景，涵蓋不同情境 reference：
- 場景 A：撰寫 JavaScript 函式 doc comment（writing-code-comments.md）
- 場景 B：起草技術文件開頭段落（writing-documents.md）
- 場景 C：設計 YAML 欄位命名（designing-fields.md）

每個場景包含：
- 明確輸入限制（只有 SKILL.md + 1 個 reference）
- 具體任務描述（可操作的寫作任務）
- 4 個評估項目（每項有明確通過條件）
- 通過判定：4 中達 3 項

量化通過標準：
- 場景通過率 100%（3/3）
- 單場景完成時間 10 分鐘內
- 評估項通過率 75% 以上

附加：4 題評估問卷（Q1-Q4，驗證 M1/M2 metric）、結果記錄表、常見失敗根因對照表。

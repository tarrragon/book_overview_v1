## 重構執行

**抽取 `_resolve_path(env_var, default)` helper**：統一環境變數優先、常數 fallback 的路徑解析邏輯。

**刪除**（4 個函式）：
- `_default_events_path()`
- `_default_annotations_path()`
- `_get_events_path()`
- `_get_annotations_path()`

**保留**（2 個模組常數）：
- `_EVENTS_JSONL_PATH`
- `_ANNOTATIONS_JSON_PATH`

**保留理由**：測試（test_dispatch_stats.py L107/109/248/512/513 等多處）透過 monkeypatch.setattr(module, "_EVENTS_JSONL_PATH", ...) 覆寫模組常數，為公開 patch point。若刪除會破壞 45 tests GREEN 硬約束。改以直接從 _PROJECT_ROOT 計算常數值（省掉兩個純計算函式），維持常數作為可被 monkeypatch 的 default 參數。

**callsites 更新**：read_events / read_annotations / write_annotations 三處改用 _resolve_path("ENV_VAR", _MODULE_CONST)，模組常數在 call time 透過模組全域查找，保留 monkeypatch 能力。

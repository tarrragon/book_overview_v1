### 1. Hook 觸發點

| 觸發點 | 行為 | 理由 |
|--------|------|------|
| `ticket track claim` PreToolUse 或 PostToolUse | 偵測 + warn / block | claim 是 PM 確認任務開始的 checkpoint |
| `ticket track complete` PreToolUse | 偵測 + warn / block | complete 是任務鏈結束的最後攔截點 |

### 2. 偵測邏輯

對目標 ticket 與其兄弟（同 parent_id）執行：

```
For each sibling_id in target.blockedBy:
    if sibling.parent_id == target.parent_id:  # 同層兄弟
        檢查 4 條件 → 任一不滿足 → 標為違規
```

### 3. 4 條件檢查規則

| 條件 | 檢查方法 | 違規訊號 |
|------|---------|---------|
| 單向（A→B，B 不依賴 A 以外的兄弟） | 兄弟集合內依賴計數 | B.blockedBy 含 ≥ 2 個兄弟 |
| 無環 | DAG 檢查 | 存在循環引用 |
| 規格→實作時序 | 依 type 啟發式判斷（ANA→IMP / DOC→IMP 視為合法） | 純 IMP→IMP 兄弟序列（誤報率高，warn-only） |
| 不可深度化 | 無法自動判斷 | 一律 warn 並要求 --acknowledge |

### 4. 行為分級

| 違規數 | 行為 |
|--------|------|
| 條件 1-2 不滿足 | block（結構錯誤可自動判定） |
| 條件 3-4 不滿足且無 --acknowledge | warn + 提示加 --acknowledge |
| 含 --acknowledge | 通過並記錄至 ticket Problem Analysis |

### 5. 邊界 case

- 缺 parent_id 或 children → skip（無兄弟結構）
- blockedBy 引用不存在 ticket → 既有 hook 已處理，不重複
- frontmatter 解析失敗 → fallback warn-only

### 6. CLI 整合

新增 `--acknowledge "<reason>"` flag 至 `ticket track claim/complete`，違規通過時將 reason 寫入 ticket Problem Analysis 章節（追蹤責任）。

#### 3. 4 條件檢查虛擬碼

```
function check_condition_1_unidirectional(target, siblings_map):
    // 對應 A1, A2, A3
    sibling_deps = [s for s in target.blockedBy if s in siblings_map]
    for dep_id in sibling_deps:
        dep = siblings_map[dep_id]
        if target.id in dep.blockedBy:
            return Violation(1, BLOCK, "雙向依賴", "移除其中一向")
    if count(sibling_deps) >= 2:
        return Violation(1, BLOCK, "多兄弟依賴", "限制單一前驅")
    return PASS

function check_condition_2_acyclic(target, siblings_map):
    // 對應 A4, A5, A6 — DFS 偵測 back edge
    visited = set(); stack = set()
    function dfs(node_id):
        if node_id in stack: return true  // 環
        if node_id in visited: return false
        visited.add(node_id); stack.add(node_id)
        for dep in siblings_map[node_id].blockedBy:
            if dep in siblings_map and dfs(dep): return true
        stack.remove(node_id)
        return false
    if dfs(target.id):
        return Violation(2, BLOCK, "循環依賴", "打破環")
    return PASS

function check_condition_3_spec_to_impl(target, siblings_map):
    // 對應 A7, A8, A9 — 啟發式：ANA/DOC 應先於 IMP
    spec_types = {ANA, DOC}
    impl_types = {IMP}
    for dep_id in target.blockedBy:
        if dep_id not in siblings_map: continue
        dep_type = siblings_map[dep_id].type
        if target.type in impl_types and dep_type in impl_types:
            return Violation(3, WARN, "純 IMP→IMP 兄弟序列",
                             "確認非規格→實作關係，或加 --acknowledge")
        if target.type in spec_types and dep_type in impl_types:
            return Violation(3, WARN, "時序錯反 IMP→spec",
                             "spec 應先於 impl")
    return PASS

function check_condition_4_no_deepening(target, ack):
    // 對應 A10, A11, A12 — 純語義，必須 ack
    if ack is null or ack.strip() == "":
        return Violation(4, WARN, "無法自動驗證不可深度化",
                         "加 --acknowledge \"理由\" 確認")
    return PASS
```

#### 4. 行為分級虛擬碼（L3 Hook 入口）

```
function hook_main(ticket_id, command, ack_flag):
    target = load_ticket(ticket_id)              // L2
    if target.parent_id is null:                 // C1
        return exit(0)                            // skip
    siblings = load_siblings(target.parent_id)
    siblings_map = {s.id: s for s in siblings}
    if target.id not in [s.id for s in siblings]:
        siblings_map[target.id] = target

    violations = []
    try:
        violations.append(check_condition_1_unidirectional(target, siblings_map))
        violations.append(check_condition_2_acyclic(target, siblings_map))
        violations.append(check_condition_3_spec_to_impl(target, siblings_map))
        violations.append(check_condition_4_no_deepening(target, ack_flag))
    except Exception as e:                        // C2 fallback
        stderr("[fallback warn-only] " + str(e))
        return exit(0)

    real_violations = [v for v in violations if v.severity != PASS]
    has_block = any(v.severity == BLOCK for v in real_violations)
    has_warn = any(v.severity == WARN for v in real_violations)

    for v in real_violations:
        stderr(format_violation(v))               // 訊息 + 建議

    if has_block:                                 // B1
        return exit(2)
    if has_warn:                                  // B2
        if ack_flag and ack_flag.strip():
            persist_ack_to_problem_analysis(target.id, ack_flag)  // §6 CLI
        return exit(0)
    return exit(0)
```

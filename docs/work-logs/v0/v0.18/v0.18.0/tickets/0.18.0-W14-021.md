| # | 測試函式 | 根因 |

**根因說明**：`test_ticket_tracker.py` 測試的是舊版 CSV 式 `ticket-tracker.py` 腳本。該腳本已被 `ticket_system/` Python 套件完全取代，CLI 介面（子命令、flag）全部重寫。所有 13 個測試回傳 exit code 2（argparse unrecognized arguments），非功能 bug。

**修復策略**：**整檔 skip 或刪除**。舊 CSV tracker 已廢棄，新 ticket_system 有獨立完整測試。建議 pytestmark skip + 註解說明廢棄原因（與 W17-192 test_analytics 同 idiom）。

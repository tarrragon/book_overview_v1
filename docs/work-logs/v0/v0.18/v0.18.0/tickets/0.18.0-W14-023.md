### 具體修改內容

**修改位置**：`.claude/pm-rules/askuserquestion-rules.md` 規則 6（line 95-146）

**修改 1：在「強制要求」表格後新增「Recommended 標籤分級」子章節**

適用情境（三類，與規則 6 既有「適用範圍」對齊）：
- ANA Ticket 路由
- 重大決策路由
- Session 關鍵分歧

標籤選擇規則：

| 證據強度 | 標籤格式 | 使用時機 | description 要求 |
|---------|---------|---------|-----------------|
| 有 WRAP/多視角支撐 | `(Recommended by WRAP)` 或 `(Recommended by 多視角)` | 已完成 WRAP 四階段或多視角審查 | 可省略證據揭露（標籤自明） |
| 無充分證據的 PM 傾向 | `(My current guess)` | Claude 根據不完整資訊的初步判斷 | **必須**明示「Claude 無充分資訊優勢，此為初步猜測」 |

共同要求：
- 仍置於 options 第一位（保留 PC-066 原設計）
- 保留所有 PC-064 禁令（禁止純文字列選項）

**修改 2：更新「反模式」表格**

新增反模式：
- 「有 WRAP 支撐卻標 `(Recommended)` 而非 `(Recommended by WRAP)`」→ 修正為明確標示證據來源
- 「無 WRAP 但用 `(Recommended)` 暗示權威」→ 修正為 `(My current guess)` + description 揭露

**修改 3：更新「理由」章節**

補充：標籤分級讓用戶區分「Claude 有證據支撐的建議」vs「Claude 初步傾向」，符合規則 5 權力不對等下的透明化要求（規則 5 §5.6 機制 2：預設答案透明化）。

**修改 4：更新版本號與 Last Updated**

- Version 升級（PATCH 或 MINOR，視既有版本決定）
- Last Updated 改為 2026-04-18
- Change Log 新增條目：「新增 Recommended 標籤分級（Phase A / W14-023 / 019.3 方案 G）」

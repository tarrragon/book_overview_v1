### 《摩擦計畫》核心概念（摘要）

Robert I. Sutton & Huggy Rao 的 *The Friction Project* 強調組織中的摩擦力是中性工具，應該：
- **降低摩擦力（friction-reducing）**：流程標準化、減少不必要的審批、高頻低風險動作自動化 → 讓優秀工作輕鬆完成
- **增加摩擦力（friction-adding）**：高風險/不可逆動作、道德倫理邊界、大決策前的停頓 → 讓糟糕工作難以發生
- 診斷工具：流程審計（process audit）、摩擦點識別（friction point identification）、旅程地圖（journey mapping）

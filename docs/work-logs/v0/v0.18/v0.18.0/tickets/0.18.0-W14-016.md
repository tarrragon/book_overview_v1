#### 根因分析

1. W14-004 為消除 hook 與 skill 中 TERMINAL_STATUSES 重複宣告，讓 hook 直接 import `ticket_system.lib.constants`
2. 但 `ticket_system.lib.__init__` 會 eager-import 整個 package（包含 ticket_loader/parser），parser 需 yaml
3. acceptance-gate-hook 以系統 Python（非 uv tool）啟動，系統 Python 無 yaml

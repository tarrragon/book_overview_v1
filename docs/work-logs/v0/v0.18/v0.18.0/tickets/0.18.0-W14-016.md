### 實作摘要

採用最小範圍 H（結構重構）：constants 上移至 package 頂層

#### 變更檔案（3 個）

| 檔案 | 操作 | 目的 |
|------|------|------|
| .claude/skills/ticket/ticket_system/constants.py | 新建 | Canonical 常數定義（package 頂層、無 yaml 依賴鏈） |
| .claude/skills/ticket/ticket_system/lib/constants.py | 改為 shim | 向後相容 `from ticket_system.lib.constants import X` |
| .claude/hooks/acceptance_checkers/children_checker.py | 改 import 路徑 | 改用 `from ticket_system.constants import TERMINAL_STATUSES` |

#### 關鍵設計決策

1. **保留 lib/constants.py 作 shim**：避免修改 30+ skill 內部使用處
2. **hook 必須用 canonical path**：lib/__init__.py eager-import 鏈（ticket_loader → parser → yaml）不經過頂層 constants
3. **新增 `__all__` 清單**：明確 public API，shim 的 `from X import *` 不污染 namespace

#### __main__ guard 處理

新 ticket_system/constants.py 的 `__main__` guard 改為 inline print（不 import lib.messages），避免循環觸發 lib/__init__ eager-import。

### 變更前後對比

| 項目 | 變更前 | 變更後 |

| 行數 | `is_stale_in_progress` 38 行 + `compute_stale_minutes` 27 行 = 65 行 | `is_stale_in_progress` 23 行 + `compute_stale_minutes` 28 行 = 51 行 |
| 重複邏輯 | `_parse_started_at` × 2、tzinfo 兼容 × 2、elapsed 計算 × 2 | 集中在 `compute_stale_minutes`，`is_stale_in_progress` 純語意檢查 + 門檻比較 |
| 函式定義順序 | `is_stale_in_progress` 在前、`compute_stale_minutes` 在後 | `compute_stale_minutes` 在前（依賴順序）、`is_stale_in_progress` 在後 |
| 公開介面 | 兩函式並列、簽名相同 | 同前；無 API 變更 |
| docstring | `is_stale_in_progress` 含完整邏輯 | `is_stale_in_progress` 註明 W10-120 重構並引用 `compute_stale_minutes` |

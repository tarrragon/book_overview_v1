```
#### 5.11 監測與修訂機制（benevolent paternalism 條件要求）

**替換為**：
```
#### 5.11 監測與修訂機制（benevolent paternalism 條件要求）

規則 5 不是一次性定稿。監測指標、觀察週期、退出條件詳見 **0.18.0-W14-019.5** ANA 結論。

| 週期 | 執行者 | 主要指標 |
|------|--------|---------|
| 即時 | PM | 每次觸發 Layer 1-4 時寫 memory（type: `rule5_trigger`） |
| 月度（每月 1 號） | PM | M1-M5 觸發頻率 + M7 疲勞降權 + Q1/Q3 案例尋找 |
| 季度（每季 15 號） | PM + 用戶 | 用戶訪談 + M4/M6/M8 統計 + Q2 摩擦案例 |
| 3 個月里程碑 | PM + 用戶 | benevolent paternalism 條件 2/3 首次正式評估 |

**關鍵退出條件**：
- 連續 1 季度 Q2 > Q1×2（摩擦多於保護）→ 降級為建議

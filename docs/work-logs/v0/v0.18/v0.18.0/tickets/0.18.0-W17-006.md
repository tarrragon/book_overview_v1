### 對比表範例（供參考）
| 面向 | --parent | --source-ticket |
|------|----------|-----------------|
| 關係欄位 | parent_id + parent.children[] | source_ticket + source.spawned_tickets[] |
| Complete 阻擋 | 父被 children 阻擋 | source 不被 spawned 阻擋 |
| 使用時機 | 父必須等所有 children 完成才 complete | 衍生項獨立排程 |
| 典型場景 | 功能拆分子任務鏈 | ANA 衍生 IMP / ADJ |

### Linux 類比理由
- fork() vs clone() man page 清楚說明 flags 差異 → POSIX 量尺第 1、4 項（man 一致、無歧義）
- execve() 副作用（替換 image）在 man 明寫 → POSIX 量尺第 5 項（行為契約）
- 本修復把隱藏副作用變成顯性契約

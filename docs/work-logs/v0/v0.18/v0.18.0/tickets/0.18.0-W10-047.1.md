### 降級理由（W10-035.3 ANA 數據）

| Hook | 3d 觸發 | Action | Action 比 | 降級策略 |
|------|--------|--------|-----------|---------|
| parallel-dispatch-verification | 1586 | 0 | 0% | 候選 5：從 PostToolUse Bash 移除註冊 |
| bash-edit-guard | 1662 | ~0 | ~0% | 候選 4：危險 pattern 白名單（保留 sed -i / perl -pi） |
| acceptance-gate | 1667 | 36 | 2.2% | 候選 4：fast-path 早期判斷 ticket track complete，跳過重操作 |

### 風險與防護

- parallel-dispatch-verification 移除：並行派發失控由 dispatch-count-guard + agent-dispatch-validation 雙重防護兜底
- bash-edit-guard 白名單：保留 sed -i / perl -pi 兩個原地編輯不可逆模式；移除的重定向模式（`> *.dart` 等）多為合法產出，誤報率高
- acceptance-gate fast-path：不改變判斷邏輯，僅在 main() 開頭新增最低成本判斷，避免每次 Bash 都跑完整流程

**已驗證的事實**：

1. **light 真實使用案例 = 0**（11/11 PROP 中無真拼字修正類）
   - light 總數 = 2（PROP-001、PROP-003）

   - 非 grandfather（真實 light 使用）= 0/2 = 0%

2. **stale-draft 樣本分布**：
   - draft 總數 = 2（PROP-001、PROP-003）
   - draft age 46 天兩例
   - > 30 天 = 2，> 60 天 = 0，> 90 天 = 0
   - 兩例皆同日 proposed（2026-03-30），非自然分布

3. **W10-110 pending 對觀察純粹度的影響**：
   - W10-099 仲裁路徑：IMP-A（hook）→ IMP-B（PROP 重評）→ ANA-C（觀察）

   - **當前 light 統計受 W10-110 pending 干擾**，無法區分「真實 light 真為 0」vs「PROP-001/003 暫時標 light 拉高 grandfather 比例」

**仍未驗證的假設**：

- 假設 A：未來新增 PROP（PROP-012+）是否有真實 light 使用（拼字修正、格式調整）？樣本 0，無法判斷。

- 假設 C：draft 滯留 > 60/90 天的真實風險強度？樣本 n=2 皆 46 天，無 > 60 天案例，提醒機制設計缺乏資料基礎。

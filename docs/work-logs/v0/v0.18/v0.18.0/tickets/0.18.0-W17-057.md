### 落地範圍

### lens-configurations.md（v1.1.0 -> v1.2.0）

情境 C / D / F / G 各新增一個「basil 視角」段落，統一格式如下：

- Agent 類型: basil-writing-critic
- 掃描焦點: 對應情境的書面文字品質（三明示 / 資訊優先序 / 正面陳述）
- 檢查項目: 4 條，覆蓋 Why/Consequence/Action 三層、資訊優先序、正向錨點、隱含表達句型

各情境 basil 視角的檢查焦點差異化設計：

| 情境 | 掃描焦點 | 特色檢查項目 |
|------|---------|------------|
| C 架構方案評估 | 架構評估文件書面品質 | 方案缺點描述是否有緩解正向錨點 |
| D 功能設計評估 | 功能設計規格書面品質 | 功能限制或排除條件是否有取代方案說明 |
| F 結論/報告審查 | 分析結論報告書面品質 | 根因與結論呈現的資訊優先序（結論先於佐證） |
| G 系統設計評估 | 系統設計文件書面品質 | 設計限制約束是否有正向說明（符合設計的做法） |

### lens-prompts.md（v1.0.0 -> v1.1.0）

情境 C / D / F / G 各新增 basil 視角 prompt 模板，沿用既有 code-block 格式（4 點聚焦清單 + 報告格式說明 + 標的變數）。

注意：lens-prompts.md 原始版本情境 C / D 無任何 prompt 章節（僅有 B / F / G），本次補充 C / D 的 basil 視角 prompt。

### 向後相容性

既有 A-G 各情境原有視角段落保持不變，basil 視角為追加段落。

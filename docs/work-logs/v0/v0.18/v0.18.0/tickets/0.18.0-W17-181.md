### 問題根因

lib `handoff_utils.is_ticket_completed` (L49) 和 `is_ticket_in_progress_or_completed` (L102) 使用 `load_and_validate_ticket` -> `get_ticket_path` -> `get_tickets_dir` -> `get_project_root()`（無參數版）。子進程環境下 `CLAUDE_PROJECT_DIR` 未設 + cwd 非專案根時，`get_project_root()` fallback 到 cwd，路徑解析失敗，兩函式保守回傳 False。

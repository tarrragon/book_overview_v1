### 技術決策

#### D1：active ticket 來源 — 直接掃描 ticket 目錄（非 subprocess）

選擇：使用 `pathlib.Path.glob` 掃 `docs/work-logs/**/tickets/*.md`，以 `lib.frontmatter_parser.parse_frontmatter` 解析。

| 方案 | 評估 | 採用 |
|------|------|------|
| subprocess `ticket track list --status in_progress` | 啟動成本數百 ms，hook 級不可接受；subagent 環境可能無 PATH | 否 |
| 直接 glob + parse_frontmatter | 無外部依賴、ms 級、與 hook_utils 同層 | 是 |
| `lib.frontmatter_parser.read_ticket` | 強制 ticket_id/version/action/target/agent 欄位，新 schema ticket 會拋 ValueError | 否 |

掃描根目錄推導：以 `Path(__file__).resolve().parents` 向上尋找含 `docs/work-logs` 的祖先（hook 隨 worktree/主 repo 走皆成立）。

實際 ticket 目錄結構：`docs/work-logs/v0/v0.18/v0.18.0/tickets/*.md`（多層 v 子目錄）。glob pattern 使用 `docs/work-logs/**/tickets/*.md`（recursive）涵蓋多層情境。

#### D2：guard 整合位置 — `main()` 而非 `detect_skip_intent`

`detect_skip_intent` 為純函式（input → output），混入 I/O 違反 SRP；`main()` 已是 I/O 層。guard 在 `main()` 中於偵測到 skip_type 後、生成提醒前插入。

#### D3：cold path 性能保證

`get_active_in_progress_ticket()` 唯一呼叫點在 `if skip_type and pattern_info:` 分支內，且僅當 `skip_type == "SKIP_SA_REVIEW"`。無 skip intent 的 prompt 完全不觸發查詢（測試 `TestColdPathPerformance` 守護此契約）。

#### D4：失敗回退語意

| 情境 | 行為 |
|------|------|
| 回傳 None（無 in_progress 或全部讀取失敗） | 維持原行為，觸發提醒 |
| 拋例外 | main() try/except，例外視同 None，觸發提醒 |
| 單一 ticket frontmatter 解析失敗 | 內部 skip 該檔，續找下一個 |
| 多個 in_progress（理論不應發生） | 取 started_at 最新者 |

#### D5：guard 規則邏輯

```
若 skip_type == "SKIP_SA_REVIEW":
    active = get_active_in_progress_ticket()  # cold path
    若 active 存在:
        type = active.get("type"); phase = active.get("current_phase")
        若 type 屬於 {"DOC", "ANA"}: 靜音
        若 type == "IMP" 且 phase == "Phase 4": 靜音
    否則: 照舊觸發
否則: 照舊觸發（其他 skip 模式不受 guard 影響）
```

### 虛擬碼（Phase 3b 落地參考）

```python
def _find_repo_root() -> Optional[Path]:
    for parent in Path(__file__).resolve().parents:
        if (parent / "docs" / "work-logs").exists():
            return parent
    return None

def get_active_in_progress_ticket() -> Optional[dict]:
    try:
        root = _find_repo_root()
        if root is None:
            return None
        in_progress = []
        for path in root.glob("docs/work-logs/**/tickets/*.md"):
            try:
                fm = parse_frontmatter(path.read_text(encoding="utf-8"))
                if fm.get("status") == "in_progress":
                    in_progress.append((fm.get("started_at", ""), fm))
            except Exception:
                continue
        if not in_progress:
            return None
        in_progress.sort(key=lambda x: x[0], reverse=True)
        fm = in_progress[0][1]
        return {"type": fm.get("type"), "current_phase": fm.get("current_phase")}
    except Exception:
        return None

# main() 內，於 detect_skip_intent 後：
if skip_type == "SKIP_SA_REVIEW":
    try:
        active = get_active_in_progress_ticket()
    except Exception:
        active = None
    if active:
        t, p = active.get("type"), active.get("current_phase")
        if t in ("DOC", "ANA") or (t == "IMP" and p == "Phase 4"):
            logger.info(f"SKIP_SA_REVIEW 因 type={t}/phase={p} 靜音")
            print(json.dumps(generate_hook_output("UserPromptSubmit"), ensure_ascii=False, indent=2))
            return EXIT_SUCCESS
```

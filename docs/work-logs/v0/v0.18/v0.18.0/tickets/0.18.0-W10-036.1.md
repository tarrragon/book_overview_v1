## Problem Analysis

### 結構重組背景

原 W10-036 結構違反「兄弟無依賴」原則：

**原結構（反例）**：
```
W10-036
├── .1 (哲學層)
├── .2 (規則層，依賴 .1 引用)
├── .3 (references，依賴 .1 引用)
├── .4 (Hook 實作，依賴 .1/.2/.3/.5)
└── .5 (實務指南，依賴 .1 引用)
```

問題：.2/.3/.5 都 relatedTo .1（哲學），兄弟層級有隱藏依賴；.4 則直接 blockedBy 其他 4 個。兄弟間依賴關係沒有透過結構表達，只藏在 blockedBy/relatedTo 欄位。

**重組結構（正例）**：
```
W10-036
├── .1（新聚合父）方法論組
│   ├── .1.1（原 .1, completed）哲學層
│   ├── .1.2（原 .2）規則層
│   ├── .1.3（原 .3）references
│   └── .1.4（原 .5）實務指南
└── .2（原 .4）Hook 實作（blockedBy .1）
```

W10-036 兄弟層僅剩 .1 vs .2：方法論聚合 vs 實作，天然的 Wave 依賴透過 blockedBy 表達。

.1 內部：.1.1 作為哲學 hub 被 .1.2/.1.3/.1.4 引用，這是方法論層內部必然依賴，透過深度表達。

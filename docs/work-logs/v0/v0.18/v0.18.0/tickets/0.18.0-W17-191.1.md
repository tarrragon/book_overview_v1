### 變更摘要

**1. 新建 `.claude/hooks/pyproject.toml`**

```toml
[project]
name = "claude-hooks"
version = "0.1.0"
description = "Claude Code hook system for book_overview_v1 project"
requires-python = ">=3.10"
dependencies = ["pyyaml>=6.0"]

[dependency-groups]
dev = ["pytest>=7.0", "pyyaml>=6.0"]

[tool.pytest.ini_options]
testpaths = ["tests"]
python_files = ["test_*.py"]
python_classes = ["Test*"]
python_functions = ["test_*"]
```

設計考量：
- `dependencies` 含 pyyaml（hook 生產端執行也需 yaml）

## Solution

抽出 `_read_json_dict(path: Path) -> Optional[dict]` helper，統一「讀檔 + json.loads + isinstance(dict) 檢查」pattern。

### 變更內容

- 新增 `_read_json_dict` helper（置於路徑常數之後、`_read_git_status` 之前）
- `_read_dispatch_active`：3 行 I/O + 解析 + 檢查 → 改用 helper + 回 None 時 raise JSONDecodeError（維持 SAFE_CALL 捕獲語意）
- `_read_handoff_pending`：3 行 I/O + 解析 + 檢查 → 改用 helper + 回 None 時 return None

### 保留的錯誤語意

- `FileNotFoundError` / `PermissionError` / `json.JSONDecodeError` 繼續由 helper 向上拋，供 SAFE_CALL 捕獲走 fallback
- `_read_dispatch_active` 非 dict root 仍 raise JSONDecodeError（與原行為一致）
- `_read_handoff_pending` 非 dict root 仍 return None（與原行為一致）

### 設計決策

1. **AC4（映射檔存在性）為阻擋級**：映射檔是 AC1/AC2 檢查的前提，缺失時其他檢查無意義；其餘 AC1-AC3 為 advisory（exit 0 + stderr），與既有 wrap-decision-tripwire-hook 模式一致。
2. **Version 非回退使用 `git show HEAD`**（非 HEAD~1）：PreToolUse 階段 working tree 改動尚未 commit，比對對象應為「上次 commit 的版本」即 HEAD。
3. **SKILL version 來源為 footer `**Version**:`**（非 frontmatter）：依 ANA 決策，避免動 Claude Code runtime 解析假設；regex 從尾部 30 行提取最後一個 match。
4. **YAML parse 失敗時降級為 info log**：用戶可能正在編輯導致 YAML 暫態無效；只有映射檔錯誤才阻擋。
5. **路徑匹配支援絕對/相對**：`_normalize_rel` 嘗試解析絕對路徑為相對 project_root；fallback 到字串 endswith 比對。

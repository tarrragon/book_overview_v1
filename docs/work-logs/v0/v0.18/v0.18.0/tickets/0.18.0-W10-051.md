   - 輸出 checklist 新增項目 6（僅 ANA 顯示）

5. `.claude/hooks/tests/test_multi_view_checker.py`（新增）
   - 11 測試覆蓋所有情境

### multi_view_status Schema 設計

- field_key: `multi_view_status`
- allowed_values: `reviewed` / `skipped` / `n_a`
- required_subfields:
  - reviewed → reviewers（代理人 ID 清單）+ conclusion（結論摘要）
  - skipped  → reason
  - n_a      → reason
- 值正規化：`n/a` / `na` → `n_a`
- Source of truth: `.claude/config/ana-solution-schema.yaml`

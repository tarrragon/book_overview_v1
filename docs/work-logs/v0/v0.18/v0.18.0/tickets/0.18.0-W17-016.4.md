<!-- To be filled by executing agent -->

行為：
- Completion Time: (pending) → frontmatter completed_at ISO 時間
- Executing Agent: (pending|TBD|未指派|空) → who.current（若已填入具體 agent 則不覆蓋）
- Review Status：保留現狀（無權威資料來源）

採 regex 置換、僅處理 placeholder，避免覆蓋人工填值。

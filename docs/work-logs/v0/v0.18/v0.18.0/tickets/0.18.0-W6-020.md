### 問題根因

W6-012.9.1 修復 readmoo-adapter logger 級別期間（commit 909865a5），代理人遇 30s promise hang。根因：`const fn = logger.info` 解構丟失 `this` 綁定，setTimeout callback 中 fn() 拋出 `TypeError: Cannot read properties of undefined (reading 'currentLevel')` 被 setTimeout 吞掉，外層 Promise 因 cleanup 在 catch 之後執行而永不 reject。

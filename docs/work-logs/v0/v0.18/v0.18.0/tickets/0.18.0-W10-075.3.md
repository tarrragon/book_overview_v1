## 產出

擴充 `.claude/skills/wrap-decision/SKILL.md` 從 v1.4.0 到 v1.5.0，新增：

### 1. 觸發條件表新增一列
"個人化建議諮詢" - 用戶提問涉及個人條件差異（健康/金錢/法律/運動裝備/安全）且 AI 即將給具體建議 → Step 0 前置

### 2. 歷史教訓章節新增 PC-071 教訓段落
說明羽球諮詢三輪偏誤案例 + WRAP 缺少 Step 0 前置閘門的根本問題

### 3. 新增「0.5 資料充足度檢查（Step 0 — Widen Options 前置閘門）」完整章節
位於「0. 錨點確認」與「W — Widen Your Options」之間。涵蓋：

- **來源與核心問句**：引用 PC-071 + 核心問句「我手上的資訊夠做這個決策嗎」
- **適用時機表**：明確哪些場景強制執行 Step 0（個人條件/具體建議/高風險）哪些可跳過（客觀知識）
- **判定流程圖**：
  - 6 項檢查清單（當事人/目標/限制/環境/假設替代/後果嚴重度）
  - 4 種判定結果（充足/不足低風險/不足中風險/不足高風險）對應不同策略
- **為什麼 Step 0 必須在 W 之前**：對照表展示「表面現象」vs「實際狀況」，證明跳過 Step 0 會讓 WRAP 變成「精美的錯誤分析」
- **反模式偵測**：4 種警告信號（選項描述假設用戶差異/通常用語/群體統計/預設用戶偏好）
- **與簡化三問的關係**：說明 Claim 場景不需 Step 0（Ticket 保證資料）、個人化諮詢必須 Step 0（用戶資料不完整）
- **溯源網**：連結至 personalized-advice-rules.md（三層機制）

### 4. 更新 footer
Version 1.4.0 → 1.5.0，Source 加入 PC-071 + W10-075

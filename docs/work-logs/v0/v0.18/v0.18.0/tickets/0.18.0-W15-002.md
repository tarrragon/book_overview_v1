## 重構摘要

將 track_validate.py 內嵌的 4 欄位驗證邏輯（stub）替換為 import 共用模組 `ticket_system.validators.frontmatter_validator`，確保 CLI validate 與 PostToolUse hook 行為完全一致。

### 主要變更
- 刪除內嵌常數：VALID_STATUS、ISO_8601_PATTERN、ACCEPTANCE_ITEM_PATTERN
- 刪除內嵌 validator：_validate_status / _validate_completed_at / _validate_acceptance / _validate_who
- 刪除 _VALIDATORS 調度表
- 新增 import：ValidationIssue、validate_frontmatter（共用模組）
- 保留 CLI adapter：新 validate_frontmatter() 委派共用模組，再把 ValidationIssue 清單適配為 (ok_fields, violations) 回傳格式供 execute_validate 呈現
- 保留 _FIX_SUGGESTIONS / _suggest_fix / execute_validate（CLI 呈現層）

### Diff 統計

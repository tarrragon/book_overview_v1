# Agent Definition 結構標準

每個 .claude/agents/*.md 的主文（description 之後）必須含以下三區塊：

## 允許產出
- 明列此 agent 可產生的產出類型（檔案類別、操作類型）

## 禁止行為
- 明列此 agent 不可做的事，包含哪種檔案、哪種操作

## 適用情境
- TDD Phase 標註（Phase 0/1/2/3a/3b/4）或獨立任務類型
- 觸發條件描述

## 豁免類別
- 元文件（如 AGENT_PRELOAD.md）
- 已 DEPRECATED 的 agent（明示 DEPRECATED 標記）
\`\`\`

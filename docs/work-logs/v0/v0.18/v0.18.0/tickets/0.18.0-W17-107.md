### 變更摘要

**Stage 3「驗證根因假設」流程擴增**：
1. 原 4 步流程改為 5 步，第 2 步插入「執行假設粒度自檢」
2. 新增「假設粒度自檢清單」獨立小節（在流程後、工具選擇前）

**粒度層清單**：
- 檔案層 / 前綴層 / 模組層 / Session 層 / 跨 session 層 / 時間層 / 全域層
- 各層附「以檔案修改史為變因」+「以 session 狀態為變因」雙範例

**自查問句**：
- 我選的粒度是哪一層？
- 是否有更細或更粗的同類變因？
- 否證該粒度後是否需驗證其他粒度才完整否證假設族？

**強制動作**：每個假設至少考慮 2 個相鄰粒度層；單一粒度即進實驗 → 標記「粒度未充分」。

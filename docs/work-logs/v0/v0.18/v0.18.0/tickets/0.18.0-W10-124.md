### 修復摘要

`_has_self_check_subsection` pattern 改為前綴匹配：

```python
pattern = rf"^### {re.escape(_SELF_CHECK_HEADING)}(?:[\s（(].*)?$"
```

允許「自檢結果」後接 whitespace / 全形括號（）/ 半形括號 () 加任意補充說明；
仍保留異義保護（`### 自檢結果摘要` 因無分隔符不匹配）。

### 設計決策

- 不僅用 `\s.*` 後綴：因為 `### 自檢結果（PC-093 H3 補充）` 標題後直接接全形左括號，無 whitespace，需顯式納入 `（(`。
- 異義保護來源：分隔符必須是 whitespace 或括號其中之一，避免 `### 自檢結果摘要` / `### 自檢結果概要` 等同字根異義誤匹配。

### 自檢結果（W10-124 Dogfood）

- [x] 巢狀深度：1 層

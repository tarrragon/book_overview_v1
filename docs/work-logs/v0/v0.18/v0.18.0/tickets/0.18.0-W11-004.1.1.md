```json

```

- 改寫 dict → `os.replace(tmp, annotations.json)` POSIX atomic

### 3. 行為規格

#### 3.1 Hook 寫入 event

1. 偵測 N >= 1 衝突
2. 組裝 event dict
3. 序列化為單行 JSON + `\n`（**必須單次 `f.write(line)`** 避免多次 write 交錯）
4. 寫入前 `fcntl.flock(f, LOCK_EX)`（並發安全）
5. 寫失敗 → logger.warning + stderr（規則 4 雙通道），**不 raise**

#### 3.2 annotations.json 寫入（atomic rename）

1. 讀 annotations.json（不存在視為空 dict）
2. 修改 dict
3. 寫 tmp 檔
4. `os.replace(tmp, annotations.json)`

#### 3.3 邊界

| 情境 | 行為 |
|------|------|
| events.jsonl 不存在/空 | list/stats 輸出「尚無事件」，exit 0 |
| 父目錄不存在 | Hook 寫入時自動建立 |
| 格式錯誤行 | skip + stderr「第 N 行格式錯誤」；stats 輸出 `malformed_lines: N` |
| 重複 event_id | list 顯示首筆；不主動去重 |
| annotations 指向不存在 event_id | list 顯示 orphan；stats 不計入 |
| TP + FP = 0 | 誤報率顯示 `N/A`，禁 ZeroDivisionError |

### 3. 行為規格（Happy Path + 邊界）

#### 3.1 Hook 寫入 event（happy path）
1. Hook 偵測 N 個衝突（N >= 1）
2. 組裝 event dict，計算 prompt_hash 與每筆 prompt_excerpt
3. `open(events.jsonl, 'a', encoding='utf-8')` append 一行 JSON + `\n`
4. 寫失敗（OSError / 唯讀 / 磁碟滿）→ logger.warning + stderr（遵 quality-baseline 規則 4 雙通道），**不 raise**（維持 Hook 非阻擋）

#### 3.2 邊界：無 log 檔 / 空檔
- events.jsonl 不存在或空：list / stats 輸出「尚無事件」，退出碼 0
- 父目錄不存在：Hook 寫入時自動建立；CLI 讀取時視為空

### 2. 核心演算法虛擬碼

#### 2.1 write_event（hook 端與 CLI 共用，但 hook 端為主寫入者）

```
function write_event(events_path, event_dict):
    ensure_parent_dir(events_path)              # mkdir -p
    line = json.dumps(event_dict, ensure_ascii=False) + "\n"   # 單行 JSON
    try:
        with open(events_path, mode='a', encoding='utf-8') as f:
            flock(f, LOCK_EX)                   # 阻塞式排他鎖
            f.write(line)                       # 必須單次 write（避免交錯）
            # flock 隨 close 自動釋放
    except OSError as e:
        log.warning("write_event failed: %s", e)
        write_stderr("dispatch_stats: write_event failed: " + str(e))
        return                                  # 不 raise（Hook 非阻擋）
```

**不變式**：
- append-only，禁止 truncate / 重寫
- 一次 write 一整行（含 \n），確保並發下行邊界完整
- flock 範圍涵蓋整個 write，無 partial line

#### 2.2 read_events

```
function read_events(events_path):
    if not exists(events_path):
        return [], 0                            # (events, malformed_lines)
    events = []
    malformed = 0
    for lineno, raw in enumerate(open(events_path, 'r'), start=1):
        raw = raw.strip()
        if not raw:
            continue
        try:
            obj = json.loads(raw)
        except JSONDecodeError:
            write_stderr(f"第 {lineno} 行格式錯誤")
            malformed += 1
            continue
        # 必填欄位驗證（schema-light）
        if not has_required_fields(obj, REQUIRED_EVENT_FIELDS):
            write_stderr(f"第 {lineno} 行缺必填欄位")
            malformed += 1
            continue
        events.append(obj)
    return events, malformed
```

**重複 event_id 處理**：read 時保留所有；下游 `_dedupe_first_wins(events)` 以首見勝。

#### 2.3 write_annotations（atomic rename）

```
function write_annotations(annotations_path, annotations_dict):
    ensure_parent_dir(annotations_path)
    tmp_path = annotations_path + ".tmp." + uuid4().hex[:8]
    try:
        with open(tmp_path, 'w', encoding='utf-8') as f:
            json.dump(annotations_dict, f, ensure_ascii=False, indent=2)
            f.flush()
            fsync(f.fileno())                   # 落盤後再 rename
        os.replace(tmp_path, annotations_path)  # POSIX atomic
    except OSError as e:
        if exists(tmp_path):
            try: os.remove(tmp_path)
            except OSError: pass
        raise                                   # CLI 端，可 raise 由 main 轉 exit code
```

#### 2.4 annotate_event（atomic mutate）

```
function annotate_event(event_id, label, note=""):
    # 1. 驗證 event 存在
    events, _ = read_events(_get_events_path())
    if event_id not in {e["event_id"] for e in events}:
        raise EventNotFound(event_id)           # CLI 轉 exit 2

    # 2. 讀現有 annotations
    annotations = read_annotations(_get_annotations_path())  # 不存在視為 {}

    # 3. 冪等檢查 + 覆寫警告
    existing = annotations.get(event_id)
    new_record = {"label": label, "note": note,
                  "annotated_at": now_iso8601_utc()}
    if existing:
        if existing["label"] == label and existing.get("note", "") == note:
            return  # 冪等，不寫
        write_stderr(f"warning: 覆寫 {event_id} 既有標註 {existing['label']} -> {label}")

    # 4. 寫回（atomic rename）
    annotations[event_id] = new_record
    write_annotations(_get_annotations_path(), annotations)
```

#### 2.5 compute_stats（純函式）

```
function compute_stats(events, annotations, groupby='none', since=None):
    events = filter_by_since(events, since)
    events = dedupe_first_wins(events, key='event_id')

    valid_event_ids = {e['event_id'] for e in events}
    # 孤兒 annotation 不計
    effective_annotations = {k: v for k, v in annotations.items() if k in valid_event_ids}

    counters = {tp: 0, fp: 0, unknown: 0, unannotated: 0}
    for e in events:
        ann = effective_annotations.get(e['event_id'])
        if ann is None:
            counters['unannotated'] += 1
        else:
            counters[label_to_key(ann['label'])] += 1

    tp, fp = counters['tp'], counters['fp']
    if tp + fp == 0:
        fpr = None                               # CLI 端轉 "N/A"
        meets_threshold = None
    else:
        fpr = fp / (tp + fp)
        meets_threshold = (fpr <= 0.10)          # AC-5

    base = {
        'total_events': len(events),
        'annotated': tp + fp + counters['unknown'],
        'unannotated': counters['unannotated'],
        'true_positive': tp,
        'false_positive': fp,
        'unknown': counters['unknown'],
        'false_positive_rate': fpr,
        'meets_threshold': meets_threshold,
    }

    # 分組
    if groupby == 'agent':
        base['groups'] = group_by(events, effective_annotations, key=lambda e: e['subagent_type'])
    elif groupby == 'keyword':
        base['groups'] = group_by_flatten_conflicts(events, effective_annotations, key=lambda c: c['keyword'])
    else:
        base['groups'] = {}

    return base
```

**`_write_event_jsonl` 函式簽名與虛擬碼**：

```
def _write_event_jsonl(
    subagent_type: str,
    prompt: str,
    conflicts_detail: List[Dict[str, str]],
    logger,
) -> None:
    """寫一行 event 到 events.jsonl，flock 並發安全，失敗不 raise。"""
    if not subagent_type or not prompt:
        return                              # AC-2 早退
    if not conflicts_detail:
        return

    import hashlib, json, fcntl
    from datetime import datetime, timezone

    prompt_hash = hashlib.sha256(prompt.encode("utf-8")).hexdigest()[:16]
    ts = datetime.now(timezone.utc).strftime("%Y%m%dT%H%M%SZ")
    event_id = f"{ts}-{prompt_hash[:8]}"
    timestamp_iso = datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")

    event = {
        "event_id": event_id,
        "timestamp": timestamp_iso,
        "subagent_type": subagent_type,
        "conflicts": conflicts_detail,
        "prompt_hash": prompt_hash,
        "prompt_length": len(prompt),
    }
    line = json.dumps(event, ensure_ascii=False) + "\n"

    try:
        _EVENTS_JSONL_PATH.parent.mkdir(parents=True, exist_ok=True)
        with open(_EVENTS_JSONL_PATH, "a", encoding="utf-8") as f:
            fcntl.flock(f.fileno(), fcntl.LOCK_EX)
            f.write(line)
            # flock 隨 close 釋放
    except OSError as e:
        logger.warning("dispatch_stats: write event failed: %s", e)
        print(f"dispatch_stats: write event failed: {e}", file=sys.stderr)
        return                              # 不 raise
```

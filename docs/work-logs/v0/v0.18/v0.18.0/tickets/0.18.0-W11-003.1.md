## Problem Analysis

### 問題根因

CLI 寫入端（track_acceptance.py `_execute_batch_check_acceptance`）將 `[x]` 前綴寫入 frontmatter `acceptance` list；Hook 讀取端（acceptance_checker.py `has_acceptance_record`）卻只掃 body 關鍵字（驗收結果/Acceptance Audit Report 等 7 個 keyword）。資料源脫鉤：PM 用 CLI 勾選 5/5 後，Hook checklist 仍顯示「[WARNING] 1. acceptance 未全勾選」。

### 影響範圍

- .claude/hooks/acceptance_checkers/acceptance_checker.py（唯一需改動）
- .claude/hooks/acceptance-gate-hook.py（caller 已有 frontmatter，透傳即可）
- 測試：新增 frontmatter acceptance 全勾選場景

### WRAP 三問

- W（Widen）：候選 A 改 has_acceptance_record 同時讀 frontmatter + body（本方案）；候選 B 改 CLI 在 body 寫關鍵字讓 Hook 看見（違反 single source of truth，否決）；候選 C 完全拋棄 body 關鍵字（舊 Ticket 退場，否決）。
- A（Attain distance）：本 ticket 機會成本為 W10-011.1（文件撰寫明示性原則），但 W11-003.1 阻塞 Hook 誤報持續污染 PM token pool，優先處理。
- P（Prepare to be wrong）：最可能失敗 = 舊 Ticket 無 frontmatter acceptance list 導致退化為 False negative。防護：空 list fallback 既有 body 關鍵字邏輯。

## Solution

### 實作說明（thyme-python-developer, 2026-04-19）

修改 `.claude/hooks/acceptance_checkers/acceptance_checker.py`：

1. **新增 `_all_acceptance_items_checked(acceptance_list)` 輔助函式**
   - 判定 list 非空且所有字串項目 `.strip()` 後以 `[x]` 開頭
   - 非字串項目或部分未勾選一律回傳 False

2. **`has_acceptance_record()` 新增 `frontmatter: Optional[dict] = None` 參數**
   - 優先檢查 frontmatter `acceptance` list 全勾選 → 回傳 True（資料源對齊 CLI 寫入端）
   - 空 list / 部分勾選 / 無欄位 / None → fallback 既有 body 關鍵字掃描（保留舊 Ticket 相容）
   - 保持向後相容：未傳 frontmatter 時行為與舊版一致（純 body 掃描）

3. **`verify_acceptance_record()` caller 透傳 frontmatter**
   - `has_accept = has_acceptance_record(ticket_content, logger, frontmatter=frontmatter)`

4. **`acceptance-gate-hook.py` 未動**：既有 call chain 已把 frontmatter 傳給 `verify_acceptance_record`，透過本次修改自動傳遞到 `has_acceptance_record`。

5. **`track_acceptance.py`（寫入端）未動**：原本就寫 `[x]` 到 frontmatter acceptance list，現在讀取端已對齊。

### 資料源對齊示意

| 階段 | 寫入端 (CLI) | 讀取端 (Hook 舊) | 讀取端 (Hook 新) |
|------|-------------|----------------|----------------|
| 勾選 AC | frontmatter acceptance `[x]` | body 關鍵字（看不到）| frontmatter `[x]` 全勾選 → True |
| 舊 Ticket | body 關鍵字 | body 關鍵字 | fallback body 關鍵字（相容）|

#### v3.1 BLK-v3-1：merge_auto_extracted_block regex 與 EOF 邊界

**managed block 末端界定（正式定義）**：

auto-extracted block 起於 `<!-- auto-extracted: ... -->` 標記行，止於下列二者先到者：

1. 下一個 `^## `（H2 heading，行首）
2. EOF（section body 字串結束）

**禁止止點**：`### `（H3，子節 heading）、其他 HTML 註解、空行。子節 heading 屬 managed block 內部結構，不作為邊界。

**正式 regex pattern**：

```python
import re

AUTO_EXTRACTED_BLOCK_PATTERN = re.compile(
    r'<!--\s*auto-extracted:[^>]*-->'   # 開頭標記（含屬性）
    r'.*?'                               # 區塊內容（非貪婪）
    r'(?=^## |\Z)',                      # lookahead：下個 H2 或字串結尾
    flags=re.MULTILINE | re.DOTALL,
)
```

**flags 解釋**：

| flag | 作用 |
|------|------|
| re.MULTILINE | 使 `^` 匹配每行開頭（支援「下一個 H2」偵測） |
| re.DOTALL | 使 `.` 跨行匹配（支援區塊內含多行內容） |

**合併行為（補強 §v2.4）**：

| existing_section_body 狀態 | 匹配結果 | merge 行為 | notes |
|--------------------------|---------|-----------|-------|
| 無標記 | pattern 無匹配 | append new_markdown 到 body 尾端 | appended_new_block |
| 有單一標記且區塊到 EOF | 匹配一次，涵蓋至 \Z | 整塊替換為 new_markdown | replaced_auto_block |
| 有標記且後接 `## Other` | 匹配至 `## Other` 前 | 整塊替換，Other section 完整保留 | replaced_auto_block |
| 有多個標記（異常） | 取第一個匹配 | 替換第一個，其餘作為 warning 記錄 | replaced_auto_block + warning |

#### v3.2 BLK-v3-2：幂等比對鍵定義

**主鍵**：標記內 `sources: {id1},{id2},...` 列表（依 SOURCE_PRIORITY 排序後比對）。

**比對邏輯**：

```python
def is_idempotent_match(existing_sources: list[str], new_sources: list[str]) -> bool:
    # 依 SOURCE_PRIORITY 排序後逐項比對；sources 列表相同 → 視為同次抽取
    return sorted(existing_sources) == sorted(new_sources)
```

**chars 欄位定位**：僅為可觀察性資訊（debug/log），**不參與幂等判定**。

**理由**：

| 考量 | 說明 |
|------|------|
| source ticket 內容可能微調 | what/why 文案小改導致 chars 變動，不應觸發重新抽取 |
| 幂等本質是「來源相同視為同次」 | 使用者預期「sources 未變就不重寫」 |
| chars 變動不代表語義變動 | 格式化差異、尾空白處理等非語義變化不應破壞幂等 |

**觸發重新抽取的條件**（sources 列表變動）：

- 新增/移除 source_ticket / blocked_by / related_to
- list 欄位順序變動（依 YAML 出現順序）

**標記範例**（chars 保留作為可觀察性資訊）：

```markdown
<!-- auto-extracted: v1 | sources: 0.18.0-W17-001,0.18.0-W17-005 | chars: 1243 -->
```

**merge_auto_extracted_block 行為修訂**：

| 既有 sources | 新 sources | chars 差異 | 結果 |
|------------|-----------|-----------|------|
| [A, B] | [A, B] | 相同 | no_change_idempotent |
| [A, B] | [A, B] | 不同（1243 → 1250） | no_change_idempotent（chars 非主鍵） |
| [A, B] | [A, B, C] | — | replaced_auto_block |
| [A, B] | [A, C] | — | replaced_auto_block |

#### v3.3 BLK-v3-3：EXTRACTABLE_FIELDS MAX_ITEMS_PER_FIELD 套用範圍

**決策**：統一套用於 `is_list=True` 欄位，不採 opt-in flag。

**理由**：

| 選項 | 優點 | 缺點 |
|------|------|------|
| 統一套用（採用） | 一致性高、無額外旗標、schema 簡潔 | 個別欄位無法豁免（但 MAX_ITEMS_PER_FIELD=5 已足夠） |
| 逐欄 opt-in flag | 彈性高 | 增加 schema 複雜度、每新增 list 欄位需決策 |

**套用對象**（v2.1 EXTRACTABLE_FIELDS）：

| source_field | is_list | 套用 MAX_ITEMS_PER_FIELD |
|-------------|---------|------------------------|
| what | False | 否 |
| why | False | 否 |
| where.files | True | 是（保留前 5 項） |
| acceptance | True | 是（保留前 5 項） |

**補強 §v2.1 ExtractableFieldRule 註解**：

```python
@dataclass(frozen=True)
class ExtractableFieldRule:
    source_field: str
    target_subsection: str
    format_template: str
    is_list: bool = False    # True → 套用 MAX_ITEMS_PER_FIELD 截斷
    dedup: bool = False      # True → 與本 ticket 既有欄位合併去重
```

**截斷行為（補強 §v2.2）**：

- is_list=True 且原始項目數 > MAX_ITEMS_PER_FIELD → 保留前 N 項，ExtractedField.truncated=True
- 截斷後若 dedup=True，去重在截斷前執行（避免前 N 項含重複）
- 於 warnings 記錄：`source {id} 的 {field} 共 {total} 項，已截斷保留前 {MAX_ITEMS_PER_FIELD} 項`

**acceptance 欄位配套**：acceptance 為 list，但項目通常較短。MAX_ITEMS_PER_FIELD=5 對實務多數 ticket 充足；若未來 acceptance 普遍 > 5 項，再評估提升上限或逐欄 opt-in。

#### 2.2 extract_context_bundle(target) -> ExtractResult

```
result = ExtractResult(target_ticket_id=target['id'], status='no_source')

IF detect_self_reference(target):
    result.status = 'self_reference'
    result.skipped.append(SkipRecord(target['id'], 'source_ticket',
                                     'self_reference', ''))
    result.warnings.append(f"偵測 self-reference：{target['id']}，停止抽取")
    RETURN result   # 短路；load_ticket NOT called（S17）

source_list = _collect_source_ids(target)  # [(id, kind), ...] 依 SOURCE_PRIORITY + YAML order
result.sources_declared = len(source_list)
IF not source_list:
    result.status = 'no_source'
    result.warnings.append('本 ticket 無可抽取來源')
    RETURN result

FOR (src_id, src_kind) IN source_list:
    version = extract_version_from_ticket_id(src_id)   # BLK-3: 非 target version
    source = load_ticket(version, src_id)              # L1 Mock 邊界
    IF source is None:
        result.skipped.append(SkipRecord(src_id, src_kind,
                                         'source_missing',
                                         f'version={version}'))
        result.warnings.append(f'source {src_id} 不存在於 {version}，略過')
        CONTINUE
    result.sources_ok += 1
    FOR rule IN EXTRACTABLE_FIELDS:
        raw = _read_nested(source, rule.source_field)   # 支援 where.files
        IF raw == DEFAULT_UNDEFINED_VALUE or raw is None or raw == []:
            result.skipped.append(SkipRecord(src_id, src_kind,
                                             'source_field_undefined',
                                             rule.source_field))
            CONTINUE
        IF rule.is_list:
            items = list(raw)
            IF rule.dedup: items = _dedup_against_target(items, target)
            truncated = False
            IF len(items) > MAX_ITEMS_PER_FIELD:
                items = items[:MAX_ITEMS_PER_FIELD]
                truncated = True
                result.warnings.append(
                    f'source {src_id} 的 {rule.source_field} '
                    f'共 {len(raw)} 項，已截斷保留前 {MAX_ITEMS_PER_FIELD}')
            result.extracted.append(ExtractedField(src_id, src_kind,
                rule.source_field, rule.target_subsection, items, truncated))
        ELSE:
            result.extracted.append(ExtractedField(src_id, src_kind,
                rule.source_field, rule.target_subsection, raw, False))

_apply_limits(result)   # S9 決策：raw 累計階段套 MAX_TOTAL_CHARS
IF result.sources_ok == 0:
    result.status = 'all_sources_missing'
    result.warnings.append('所有宣告來源皆不存在')
ELIF result.sources_ok < result.sources_declared:
    result.status = 'partial'
ELSE:
    result.status = 'success'
RETURN result
```

Non-raising 保證：load_ticket / extract_version 若意外拋例外由 caller (create.py §create-insert) 捕捉降級（§v2.3 / S20）。

#### 2.3 render_context_bundle_markdown(result) -> str

```
IF result.status IN ('no_source', 'self_reference', 'all_sources_missing'):
    RETURN ''   # §v2.3 這些 status 不寫入 section

sources_sorted = sorted({f.source_id FOR f IN result.extracted})
header = f'<!-- auto-extracted: v1 | sources: {",".join(sources_sorted)} | chars: {result.total_chars_estimate} -->'
lines = [header, '## Context Bundle', '']
FOR rule IN EXTRACTABLE_FIELDS:
    fields = [f FOR f IN result.extracted IF f.target_subsection == rule.target_subsection]
    IF not fields: CONTINUE
    lines.append(f'### {rule.target_subsection}')
    FOR f IN fields:
        IF rule.is_list:
            FOR item IN f.raw_value:
                lines.append(rule.format_template.format(source_id=f.source_id, value=item))
        ELSE:
            lines.append(rule.format_template.format(source_id=f.source_id, value=f.raw_value))
        IF f.truncated: lines.append(f'  {TRUNCATE_INDICATOR}')
    lines.append('')
RETURN '\n'.join(lines)
```

#### 2.4 merge_auto_extracted_block(existing, new) -> (merged, notes)

```
# §v3.1 正式 regex（模組級常數，僅 compile 一次）
AUTO_EXTRACTED_BLOCK_PATTERN = re.compile(
    r'<!--\s*auto-extracted:[^>]*-->.*?(?=^## |\Z)',
    flags=re.MULTILINE | re.DOTALL,
)

match = AUTO_EXTRACTED_BLOCK_PATTERN.search(existing)
IF match is None:
    # 無既有標記：append 於 body 尾端，保留 PM 手寫
    sep = '\n\n' if existing and not existing.endswith('\n\n') else ''
    RETURN (existing + sep + new, ['appended_new_block'])

existing_sources = _parse_sources_from_marker(match.group(0))  # §v3.2 主鍵
new_sources = _parse_sources_from_marker(new)
IF sorted(existing_sources) == sorted(new_sources):
    # §v3.2 chars 差異非主鍵，不影響幂等
    RETURN (existing, ['no_change_idempotent'])

# sources 變更：整塊替換（§v3.1 regex 已確保止於下一個 H2 或 EOF，H3 不作邊界）
merged = existing[:match.start()] + new + existing[match.end():]
notes = ['replaced_auto_block']
IF AUTO_EXTRACTED_BLOCK_PATTERN.search(existing, match.end()):
    notes.append('warning: multiple auto-extracted markers detected')
RETURN (merged, notes)
```

## Problem Analysis

### Context Bundle（PM 前台 Phase 1 盤點 2026-04-19）

#### 現況：FORBIDDEN_KEYWORD_MAP 既有 6 類

- 實作（6 條 regex）
- 修改檔案（3 條）
- git commit（2 條）
- 設計功能規格（2 條）
- 直接執行測試修復（2 條）
- 執行測試（3 條）

共 18 條 regex，主要覆蓋 W5-001 sage 越界案例（prompt 要求 sage 寫 .py）。

#### 新 Pattern 類別候選（依 agent 禁止行為頻次）

**類別 A — Ticket CLI 操作**（覆蓋 rosemary / incident-responder 禁止直接執行）
- keyword: `ticket CLI`
- patterns:
  - `ticket\s+(?:track|create|migrate|handoff)\s+\w+`
  - `/ticket\s+\w+`

**類別 B — 規格/文件修改**（覆蓋 lavender / oregano / star-anise 禁止越權）
- keyword: `修改規格`
- patterns:
  - `(?:修改|編輯|Edit)[^\n]{0,20}(?:spec|規格|需求|use-?case)`
  - `Edit[^\n]{0,30}\.md`（配合 context 判斷）

**類別 C — Git 寫入（非 commit）**
- keyword: `git 寫入`
- patterns:
  - `git\s+(?:push|merge|rebase|cherry-pick|reset\s+--hard)`
  - `(?:推送|合併|變基)[^\n]{0,10}(?:分支|branch|PR)`

**類別 D — 重構/移除**（覆蓋 linux 「只分析不修改」、pepper「只規劃不實作」）
- keyword: `執行重構`
- patterns:
  - `(?:執行|進行)[^\n]{0,10}重構`
  - `(?:移除|刪除)[^\n]{0,20}\.(?:py|js|ts|dart|go)`
  - `\brefactor\b`（配合 Edit 動詞）

**類別 E — 系統級審查**（覆蓋 lavender 禁止系統級審查）
- keyword: `系統審查`
- patterns:
  - `系統級?審查`
  - `(?:盤點|審計)[^\n]{0,10}(?:系統|架構|全專案)`

**類別 F — 分支/worktree 操作**
- keyword: `分支操作`
- patterns:
  - `git\s+(?:checkout|branch|switch)`
  - `worktree\s+(?:add|remove|prune)`

派發 thyme-python-developer 執行：

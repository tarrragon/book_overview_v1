| 訊號 ID | 類別 | 觸發來源 | W15-009 對應 |

### W15-009 實作範圍修正

原設計「4 個 Hook 觸發點」調整為：
- **擴充** S3 `ana_claim` 訊息（T7 吸收）
- **新增** S4 `reflection_depth_challenge`（T6）
- **不建 Hook**：T5 已由 S1 覆蓋；T3 由 SKILL 主動呼叫（Phase 4 評估 flow 內）
- **新增 config 類別**：`reflection_triggers` 與 `wrap_tripwire` 並列

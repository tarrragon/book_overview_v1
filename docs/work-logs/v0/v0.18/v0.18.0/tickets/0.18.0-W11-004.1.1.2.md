## 重構執行

**扁平化 annotate_event**：移除 overwrote_different 中間旗標與嵌套 if。

**改動前**：
- 嵌套 if（冪等檢查在 existing 分支內部）
- overwrote_different 中間旗標隨分支條件改寫

**改動後**：
- 冪等早退（existing 存在且 label/note 相同 → return False）
- overwrote 單一 expression 計算：bool(existing) and existing.get("label") != label
- 線性流程：冪等檢查 → 計算 overwrote → 寫入 → return

**語意保持**：
- 冪等行為不變（同 label + 同 note → 不寫入，回 False）
- overwrote 回傳語意不變（既有標註存在且 label 不同才回 True）
- note 變更但 label 相同的情形：原版 overwrote_different=False（因為只在 label 不同時設 True），會寫入但回 False；新版 existing.get("label") != label 為 False，overwrote=False，行為一致。

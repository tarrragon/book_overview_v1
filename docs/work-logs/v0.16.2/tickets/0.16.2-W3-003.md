### SyncMetadataManager 功能
- 管理 chrome.storage.sync 元資料（設定/時間戳/版本號）
- 依賴注入設計：logger、eventBus、chromeStorage 皆可注入
- _validateSize() 寫入前檢查 JSON 不超過 8,192 bytes
- _handleStorageChanged() 只處理 areaName=sync 且相關 key
- _storageSet/_storageGet 封裝 callback API 為 Promise
- 錯誤處理：超配額 VALIDATION_ERROR、API 失敗 OPERATION_ERROR
- 匯出常數：STORAGE_KEYS、QUOTA_BYTES_PER_ITEM、SYNC_METADATA_EVENTS

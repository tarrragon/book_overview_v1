## 🛠 技術實作策略

### 核心遷移模式

**StandardError → ErrorCodes IIFE模式**:
```javascript
// 舊模式 (已棄用)
throw new StandardError('ERROR_CODE', 'message', { details })

// 新模式 (標準)
throw (() => {
  const error = new Error('message')
  error.code = ErrorCodes.ERROR_CODE
  error.details = { details }
  return error
})()
```

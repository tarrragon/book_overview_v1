## 🎯 任務目標

### 清理背景
在 v0.13.x 版本的 StandardError 系統遷移和 ESLint 合規性修復過程中，產生了大量的：
- 備份檔案（.backup.*, backup_*, .backup-*）
- 修復腳本（fix-*.sh, *-cleanup*.sh, *-fix.js）
- 臨時工具檔案（quick-*.js, targeted-*.js, batch-*.js）

這些檔案在修復完成後已不再需要，影響專案檔案結構的清潔度。

### 清理目標
- 移除所有不再需要的備份檔案
- 清理修復過程中產生的臨時腳本
- 保留可能需要的工具腳本（如 fix-mode-complete.sh）
- 確保清理後專案功能完整無損

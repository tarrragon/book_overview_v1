### Context Bundle

**改動範圍**：
- `src/data-management/BookSchemaV2.js:47` — 移除 `platform: 'READMOO'` 頂層硬編碼
- `src/data-management/BookSchemaV2.js:66` — `source` 欄位改為 `required: true`，移除 `default: 'readmoo'`
- 連帶影響：`applyDefaults()` 不再為缺少 source 的書籍自動填入 'readmoo'

**影響範圍分析**：
- `BOOK_SCHEMA_V2.platform` 無外部消費者直接讀取（grep 確認）
- `source` 欄位改 required 後需確認 `tests/unit/data-management/book-schema-v2.test.js` 無硬編碼 readmoo 預期
- `readmoo-platform-migration-validator.js` 使用的 `platform: 'READMOO'` 是獨立常數非 schema 引用，不受影響
- 既有 Readmoo 書籍已有 source='readmoo'；博客來 adapter 已提供 source='books-com-tw'

**新增需求**：
- 匯出 `PLATFORM_NAMES` 列表（從 PlatformRegistry 導出或 schema 內建）供驗證用
- 考慮向後相容：已存在無 source 的舊書籍需在 migration 中補 'readmoo'

### WRAP 三問

**W**：(1) source 改 required + enum 受控 (2) source 保持 optional 但移除 default — 選方案 1 更嚴謹

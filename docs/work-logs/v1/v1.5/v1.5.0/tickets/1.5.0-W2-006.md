## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->

CJS module `src/content/platform/platform-registry.js`，匯出 `PLATFORM_CONFIGS / detect / getAllMatchPatterns / getRegisteredPlatforms`。

- `PLATFORM_CONFIGS`：初始一筆 Readmoo 配置（name / displayName / matchPatterns / hostnames / adapterFactory / libraryUrl）。adapterFactory 為 lazy require `() => require('../adapters/readmoo-adapter')`。
- `detect(url)`：解析 hostname（解析失敗回 null），以完全相等或 `.`-邊界後綴比對命中書城，回傳 `{ config, createAdapter }`；`createAdapter(options)` 呼叫 adapterFactory 建構適配器。後綴邊界避免 `evilreadmoo.com` 誤命中 `readmoo.com`。
- `getAllMatchPatterns()`：flatMap + Set 去重所有書城 match patterns（供 manifest content script 註冊）。
- `getRegisteredPlatforms()`：回傳配置清單副本（slice），防止外部修改內部狀態。

新增書城只需在 `PLATFORM_CONFIGS` 增一筆配置（開放封閉原則）。

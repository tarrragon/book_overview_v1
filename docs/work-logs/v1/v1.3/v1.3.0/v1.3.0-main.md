## 技術筆記

### hooks/lib 合併策略

合併後統一模式：所有 hook 的 sys.path 指向 `.claude/`，import 統一 `from lib.xxx`。未建兼容層（原 AC3），原因是 Python package shadowing：hooks/lib/__init__.py 存在時會遮蔽 .claude/lib/，導致所有 import 失敗。

### identity-guard --as 使用率分析

255 筆樣本統計（W1-082 cutoff 後）：exempt 121 / warn 68 / pass 62 / deny 4。代理人 complete 維度使用率 92.5%，整體被 PM bookkeeping（check-acceptance/set-acceptance 未帶 --as）稀釋至 49.3%。4 筆 deny 全為 PM UX 摩擦，非代理人越權。

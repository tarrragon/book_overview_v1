## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->

新增 `.claude/lib/test_result_parser.py` 共用模組：

- `detect_framework(data)`: 自動辨識 pytest 或 jest JSON 格式
- `parse_test_json(json_input)`: 解析 JSON 字串或 dict，回傳 `TestSummary`
- `TestSummary` dataclass: `passed`/`failed`/`skipped`/`total`/`is_green`
- pytest `error` 計入 `failed`（合併計數）
- jest `numPendingTests` 對應 `skipped`

不需新增 `pytest-json-report` 依賴，parser 直接解析其輸出格式。

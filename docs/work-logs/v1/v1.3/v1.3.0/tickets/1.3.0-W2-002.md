## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->

在 `version.py` 新增 `check_version_all_completed(version)` 函式，檢查指定版本所有 ticket 是否皆為 TERMINAL_STATUSES（completed / closed），並透過 `_find_next_active_version()` 從 todolist.yaml 取得下一個 active 版本。

在 `track_query.py` 的 `execute_summary` 和 `track_dashboard.py` 的 `dashboard_main` 中各加入 `_print_all_completed_warning()` helper，於輸出結尾 emit warning。

修改檔案：
- `.claude/skills/ticket/ticket_system/lib/version.py` — 新增 `check_version_all_completed` + `_find_next_active_version`
- `.claude/skills/ticket/ticket_system/lib/command_tracking_messages.py` — 新增 3 則 warning 訊息常數
- `.claude/skills/ticket/ticket_system/commands/track_query.py` — 新增 `_print_all_completed_warning` + 在 `execute_summary` 呼叫
- `.claude/skills/ticket/ticket_system/commands/track_dashboard.py` — 新增 `_print_all_completed_warning` + 在 `dashboard_main` 呼叫

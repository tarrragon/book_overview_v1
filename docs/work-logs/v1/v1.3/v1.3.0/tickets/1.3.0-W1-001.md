## Solution
在 `.claude/references/skill-marketplace-standard.md` 建立 Skill 獨立上架標準，涵蓋：

1. **設計方向**（§1）：單一職責判準表、環境解耦禁止/替代表、分類軸（依專案類型非語言）
2. **獨立性設計原則**（§2）：companion 缺席降級寫法表、跨 skill 依賴分級表、不可拆分三問判準
3. **落地層架構**（§3）：通用層/落地層分離（project-integration/ 不隨上架）
4. **上架前檢查清單**（§4）：環境無關性、獨立性、分類術語、結構品質四類
5. **對照案例**（§5）：wrap-decision 通用層設計 + 落地層分離 + 缺席降級示範

同步在 skill-design-guide §13 延伸閱讀加入交叉引用。

## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->
實作 Popup 書庫跳轉選單，採選單式架構預留多書城擴充，v1.4.2 僅含 Readmoo。

**改動檔案**（已於 commit bc2bf9017 落地於 feature branch）:
1. `src/popup/constants/bookstore-config.js`（新增）— BOOKSTORE_LIST SSOT，Object.freeze 凍結，每筆 { id, name, url, enabled }。新增書城僅需追加 entry。
2. `src/popup/constants/index.js`（修改）— 匯出 bookstore-config。
3. `src/popup/constants/ui-text.js`（修改）— 新增 NAVIGATION_TEXT（SECTION_TITLE '前往書庫', GO_BUTTON_ARIA_PREFIX '前往 '）。
4. `src/popup/components/ui-factory.js`（修改）— 新增 createBookstoreNavSection 工廠函式：依 enabled 過濾，reuse createButton（secondary variant），class 複用 status-card / results-header / action-buttons，每按鈕帶 data-bookstore-id 與 aria-label。
5. `src/popup/popup.html`（修改）— header 下方新增掛載點 <div id="bookstoreNavMount">。
6. `src/popup/popup.js`（修改）— 新增 mountBookstoreNav()，初始化時掛載選單；onNavigate 以 chrome.tabs.create({ url: store.url }) 跳轉；含 mount/uiFactory/BOOKSTORE_LIST 存在性 guard。
7. 測試（新增）— tests/unit/popup/bookstore-config.test.js、bookstore-nav.test.js、navigation-text.test.js。

**設計符合規範**:
- 書城清單為 config SSOT，UI 邏輯不硬編碼（project-conventions）。
- UI 文字進 ui-text.js，元件用 ui-factory 工廠，CSS 複用 design-system class（不新增 CSS）。
- TDD：測試先行（RED），實作後 GREEN 全綠。

**收尾發現**: 接手時 src/test 已由前一執行體 commit（bc2bf9017）落地於 feature branch（git status 僅 ticket md 待 log），屬 record plane 落後 world plane（tool-output-trust 規則 5）。本次補完 ticket 日誌、驗證測試 GREEN、收尾 acceptance 與 complete。

## Context Bundle

### 規格

- Readmoo 書庫 URL：`https://read.readmoo.com/#/library`（來源：`docs/bookstores/readmoo.md`）
- v1.4.2 僅含 Readmoo 一個書城，但架構預留多書城擴充（v1.5+）
- 選單放在 header 下方、主操作按鈕上方（或主操作卡片內頂部）

### 實作策略

1. **新增 `src/popup/constants/bookstore-config.js`**（書城清單 SSOT）：
   ```js
   const BOOKSTORE_LIST = Object.freeze([
     { id: 'readmoo', name: 'Readmoo', url: 'https://read.readmoo.com/#/library', enabled: true }
   ])
   ```
   新增書城只需在此陣列加一筆 entry。

2. **`src/popup/constants/ui-text.js`** 新增 NAVIGATION_TEXT 分類：
   ```js
   const NAVIGATION_TEXT = Object.freeze({
     SECTION_TITLE: '前往書庫',
     GO_BUTTON_ARIA_PREFIX: '前往 '  // + bookstore name
   })
   ```

3. **`src/popup/components/ui-factory.js`** 新增 `createBookstoreNavSection` 工廠函式：
   - 接收 bookstore list，回傳含按鈕列的 DOM section
   - 每個書城一個 secondary button（reuse 既有 createButton）
   - class 複用 `status-card`（次階 panel）

4. **`src/popup/popup.html`** header 下方新增掛載點 `<div id="bookstoreNavMount"></div>`

5. **`src/popup/popup.js`** 初始化時：
   - import bookstore config + ui-factory
   - 建立選單 DOM 掛載到 `#bookstoreNavMount`
   - click handler：`chrome.tabs.create({ url: bookstore.url })`

### Commit Policy

- 單一 commit 到 feature branch `feat/1.4.2-W1-001-bookstore-nav`
- commit message：`feat(popup): add bookstore navigation menu (1.4.2-W1-001)`

### 必讀規範

- `docs/project-conventions.md`（寫 src 前必讀）
- `src/popup/constants/ui-text.js` 分類軸為功能領域

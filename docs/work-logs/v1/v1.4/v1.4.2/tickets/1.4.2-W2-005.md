### PM 前台分析進度（2026-06-24）

#### 已確認事實

1. `readmoo-adapter.js:1071` 提取封面邏輯：`img.getAttribute('src') || ''`
2. 只讀 `src` 屬性，**未處理** `data-src` / `data-original` / `srcset` 等 lazy load 屬性
3. Readmoo 書庫是 React SPA + 虛擬捲動（`#react-container`）
4. adapter 有 `scrollAllIntoView`（L695）捲動觸發載入機制
5. 用戶 DOM 貼文顯示兩本書的 `<img class="cover-img">` 都有完整 `src`，但實機結果部分書封面缺失

#### 待驗證假設（需實機觀察）

- H1：React 虛擬捲動回收——捲動過的書被 React unmount 後 DOM 消失，重新渲染時 img src 延遲設定
- H2：CDN 圖片 src 延遲——img 先建空 DOM 再異步設 src，提取時 src 為空
- H3：部分書籍根本沒有 cover-img 元素（DOM 結構不同）
- H4：Readmoo 用 IntersectionObserver lazy load，viewport 外的 img src 尚未載入

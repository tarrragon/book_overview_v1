## Context Bundle

### 實作策略

content-coordinator.js L579 的 `chrome.scripting.executeScript` 缺少防護。
同檔 content-script-coordinator-service.js L386 已有正確範例：`if (typeof chrome !== 'undefined' && chrome.scripting) {`

修復：L579 前加同樣防護：
```js
if (typeof chrome === 'undefined' || !chrome.scripting) {
  this.logger.warn('[SKIP] chrome.scripting 不可用，跳過重連')
  contentScript.status = CONTENT_SCRIPT_STATES.ERROR
  return
}
```

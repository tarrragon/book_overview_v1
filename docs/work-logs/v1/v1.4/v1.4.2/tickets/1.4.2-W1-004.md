# Execution Log

## Task Summary

評估 Popup 狀態提示用語面向一般使用者（移除開發者術語如 Content Script）

---

## Problem Analysis
<!-- Schema[ANA/Problem Analysis]: 必填（.claude/pm-rules/ticket-body-schema.md） -->
### 問題根因
popup 多處 hardcoded 開發者術語（Content Script / Background Service Worker / 擴展）作為使用者可見狀態文字。實作時直接寫技術術語，未區分「使用者可見狀態」與「開發者除錯資訊」。

### 影響範圍
- src/popup/popup-event-controller.js（updateStatus 三處）
- src/popup/constants/ui-text.js（STATUS_TEXT / ACTION_TEXT / ERROR_TEXT / PAGE_INFO_TEXT）
- src/popup/popup.js（使用者狀態文字 + 錯誤常數）
- src/popup/popup.html（7 處「擴展」，初次盤點漏，實機補抓）

### 相關 Error Pattern
無直接相關。盤點缺口教訓：用語盤點須涵蓋 HTML 顯示文字，非僅 .js。

## 重現實驗結果
## 盤點方法
grep popup/ 所有 updateStatus() 顯示文字 + ui-text.js STATUS_TEXT + popup.js 錯誤訊息常數，對照本 session 實機 popup snapshot 看到的文字。

## 盤點結果（面向使用者顯示的開發者術語）

| 現顯示文字 | 位置 | 分類 |
|-----------|------|------|
| Content Script 連線正常 | controller L399 | 開發者術語（嚴重）|
| Content Script 載入中 | controller L405 / ui-text CONTENT_SCRIPT_LOADING | 開發者術語 |
| Background Service Worker 連線正常 | controller L333 | 開發者術語 |
| 線上 — 背景服務連線正常 | popup.js L758 | 偏技術 |
| Content Script 尚未就緒 / 連線失敗 | popup.js L161-162（錯誤訊息）| 開發者術語 |
| 擴展設定/狀態/就緒/重新載入擴展 | 多處 + ui-text | 地區用語（extension 直譯，台灣慣用「擴充功能」）|

## 實機佐證
本 session popup snapshot 多次顯示「Content Script 連線正常」「Content Script 載入中」，確認用戶內測回報屬實——一般使用者無法理解 Content Script / Service Worker。

## 已驗證 vs 未驗證
- 已驗證：上述術語確實顯示於 popup UI（grep + 實機 snapshot 雙確認）
- 範圍：主要在 popup-event-controller.js updateStatus 三處 + ui-text.js 2 條 + popup.js 錯誤常數 2 條 + 「擴展」散落多處

## Solution
<!-- Schema[ANA/Solution]: 必填（.claude/pm-rules/ticket-body-schema.md） -->
### WRAP 檢驗

- Widen：三選項 (a) 全改寫使用者友善 (b) 技術細節移診斷模式 (c) 維持現狀
- Reality：用戶內測回報 + 本 session 實機 snapshot 雙確認，非臆測
- Attain：改寫建議表如下，具體可執行
- Premortem：過度簡化恐丟失除錯資訊 → 技術細節保留於診斷模式（已有 ACTION_TEXT.DIAGNOSTIC），主狀態改使用者友善

### 改寫建議

| 現文字 | 改寫建議 |
|--------|---------|
| Content Script 連線正常 / 可以開始提取書庫資料 | 已連線到書庫頁面 / 可以開始提取 |
| Content Script 載入中 / 請稍候或重新整理頁面 | 正在連線書庫頁面 / 請稍候或重新整理頁面 |
| Background Service Worker 連線正常 / 系統就緒 | 擴充功能已就緒 / 系統就緒 |
| 線上 — 背景服務連線正常 | 擴充功能已就緒 |
| Content Script 尚未就緒 / 連線失敗（錯誤）| 無法連線書庫頁面，請重新整理頁面後再試 |
| 擴展（設定/狀態/就緒/重新載入擴展）| 擴充功能（台灣慣用，全域替換）|

### 決策
採 (a)+(b) 混合：主狀態改使用者友善用語；「擴展」→「擴充功能」全域替換；原始技術細節（Content Script / Service Worker 連線狀態）保留於診斷模式供進階除錯。

### Spawn 落地
spawn IMP child 執行文字替換（popup-event-controller.js updateStatus 三處 + ui-text.js + popup.js 錯誤常數 + 「擴展」全域），派 thyme-extension-engineer 實作、basil-writing-critic 審改寫文字品質。

## Context Bundle

### 規格

- 表格 header 移除「標籤」欄，COLUMNS 從 8 改 7
- 作者欄 CSS 拓寬（min-width 或 flex 調整）
- 每行末尾加展開箭頭（toggle button）
- 點擊展開 → 行下方插入 colspan=7 的展開行，內含 tagCellRenderer.createTagCell() 的結果
- 展開行以 chip/badge 列呈現標籤（複用既有 tag badge CSS）
- 再次點擊收合（toggle）

### 實作策略

1. **overview.html L73**：移除 `<th>標籤</th>`

2. **overview-page-controller.js**：
   - CONSTANTS.TABLE.COLUMNS: 8 → 7（L59）
   - createBookRow（L775-860）：
     - 移除 L855-857 的 tagCell 直接 append
     - 狀態欄後新增展開按鈕欄（`<td>` 含 toggle button）
     - toggle button click → 插入/移除展開行（`<tr class="tag-expand-row"><td colspan="7">...</td></tr>`）
     - 展開行內放 `tagCellRenderer.createTagCell(book.tagIds || [])`
   - 用 `row.dataset.expanded = 'true'/'false'` 追蹤展開狀態

3. **overview.css**：
   - 作者欄 `.book-author-cell` 加 `min-width: 100px`
   - 新增 `.tag-expand-row td` 樣式（padding、background 區隔）
   - 新增 `.expand-toggle-btn` 樣式（小箭頭按鈕）
   - 展開行 transition（height/opacity）

4. **tagCellRenderer.createTagCell**：無需修改，直接複用

## Context Bundle

### 實作策略

popup-ui-components.js L31 的 `require('src/popup/components/ui-factory')` 在 Chrome runtime 失敗（不走 esbuild bundle）。

修復方案：將 require 包在 typeof 防護中，fallback 到 window global：
```js
let uiFactoryExports
if (typeof require !== 'undefined') {
  try { uiFactoryExports = require('src/popup/components/ui-factory') } catch (e) { /* bundle 外 */ }
}
if (!uiFactoryExports && typeof window !== 'undefined' && window.uiFactory) {
  uiFactoryExports = window.uiFactory
}
const { createStatusIndicator, createProgressSection, createResultsSection, createErrorSection } = uiFactoryExports || {}
```

同時確認 ui-factory.js 在 popup.js bundle（esbuild IIFE）中已有 window 注入（popup.js 的 window.uiFactory 或類似機制）。若無，需在 ui-factory.js 底部加 window 注入。

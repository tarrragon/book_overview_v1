## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->
### 根因
`load_preserve_list` 的 PyYAML 路徑已正確 fail-loud（stderr + raise），但測試環境以 `--with pytest`（無 pyyaml）執行，`yaml is None` 落入簡易 fallback 解析器。fallback（原 line 249-265）對 malformed 輸入靜默忽略並回傳空集合，未 fail-loud——違反 observability 規則 1 / quality-baseline 規則 4。

### 修改
僅改 `load_preserve_list` 的 fallback 解析器（單檔單函式）：偵測簡易解析器無法可靠處理的兩種結構，改為雙通道 fail-loud（stderr 警告 + raise ValueError）：
1. tab 縮排（YAML 語法錯誤）。
2. inline flow 清單（`preserve: [...`）。

未動 PyYAML 路徑、未動其他函式。

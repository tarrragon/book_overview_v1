## 重現實驗結果
### 實驗方法（可重現，無 chpwd 污染）

關鍵：用 `git -C "$TMP"`（不 cd、不觸發 zsh chpwd hook 的自動 ls）；初次用 `(cd "$TMP" && ...)` 致 ls 輸出混入排序檔，comm 產生假數據（誤報 2610）。

```bash
TMP=$(mktemp -d)
git clone --depth=1 https://github.com/tarrragon/claude.git "$TMP"
git -C "$TMP" ls-files --stage | awk '$1!="160000"{print $4}' | sort > remote.txt   # 排除 gitlink
git ls-files .claude/ | sed 's|^\.claude/||' | sort > local.txt
comm -23 remote.txt local.txt   # 遠端有 + 本地 tracked 無 = 孤兒
```

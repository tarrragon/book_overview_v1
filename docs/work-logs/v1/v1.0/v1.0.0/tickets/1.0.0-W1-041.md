## Solution
<!-- Schema[ANA/Solution]: 必填（.claude/pm-rules/ticket-body-schema.md） -->

### 裁決

宿主機制收斂：**事發 session 自身進程內的同 turn 第二執行流（in-process duplicate turn execution）**。外部三候選（SDK / 獨立進程 resume-replay / workflow 內部）以正向物證全排除；同進程內細分（通知重投 vs turn fork）依證據邊界明確標記不可知。完整事件圖像：通知 turn 44875861 在同一進程內存在兩條執行流——存續流走 Workflow（記錄持久化），ghost 流走 6 個背景 Agent + 2 個背景 bash（副作用持久化、主迴圈記錄被單寫者仲裁淘汰），ghost 流持續 22 分鐘完成完整主迴圈（派發 → 消費 → 修復 commit → append → complete → spawn）。

### 實作策略（語言無關，thyme 轉 JS）

於既有 cross-device-round-trip.test.js 新增 v3 group（鏡像既有 Group A/B/C 結構與 helper 風格）：

1. 新增 helper `exportToV3Json(books)`：`new BookDataExporter(books).exportToJSON({ formatVersion: '3.0.0', pretty: false })`（v3 不需 tags/tagCategories，everything-as-tags）。
2. round-trip 鏈：`exportToV3Json` → 既有 `simulateCloudTransport`（純字串 parse/stringify）→ 既有 `parseImportContent`（ContentParser，已自動路由 canonical）。
3. 斷言無損（acceptance 2）：
   - 還原後 books 數量 = 原始；逐本 id/title/progress/readingStatus/tagIds 對齊。
   - everything-as-tags：原 book 的 authors/source/tagIds 經 canonical 往返後正確還原（用 mapCanonicalToV1Book 的 converge 邏輯）。
   - `_passthrough` 欄位保留（rebuildFromPassthrough）：放入含非標準欄位的書，往返後該欄位不丟失。
4. fixture：可復用既有 device fixture 或自建含 tag/_passthrough 的小樣本。

### acceptance #2：clean_stale_files 完整邊界（push + pull 兩端 × 兩路徑）

| 端 / 路徑 | 觸發 | 行為 | 對 27 孤兒 |
|-----------|------|------|-----------|
| Push --clean | opt-in | 遠端有 + 本地 staging 無 → unlink + push deletion | 能清 |
| Push 常規 | 無 --clean | 不清遠端 | 殘留 |
| Pull 3-way merge | base SHA 可達 | 只套 base→HEAD delta | 不帶回 |
| Pull full overlay | base 無/不可達 | sync_directory 全量覆蓋本地 | 帶回（孤兒複製到本地） |

**安全防護不對稱**：push clean_stale_files 有 should_exclude 但無執行前 dry-run 預覽；pull full overlay 有 preview_overlay_changes（**print-only，無互動確認**——原 ANA 誤判有互動確認，由仲裁修正）+ git 追蹤感知。

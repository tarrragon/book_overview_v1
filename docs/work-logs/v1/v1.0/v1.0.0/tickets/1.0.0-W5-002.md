### Round-trip 驗證結果

**驗證流程**（W5-001 五步驟）：
1. V1 匯出 canonical v3 JSON（6 本書，六態 readingStatus）→ fixture
2. APP 匯入 V1 fixture → APP 匯出 canonical v3（7/7 tests passed）
3. V1 匯入 APP canonical → mapCanonicalToV1Book 還原
4. Diff 比對原始 V1 books vs round-trip 還原
5. 成功標準逐項驗證

### 成功標準結果

| 標準 | 結果 | 說明 |
|------|------|------|
| SC-1 | PASS | 6 本書 id/title/source 完全無損 |
| SC-2 | PASS（含 normalize） | 六態 readingStatus 經 normalize reading_status→readingStatus 後雙向映射正確 |

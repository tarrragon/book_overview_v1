### 變更摘要（tool-output-trust-rules.md，1.0.0 → 1.1.0）

| 2 | 與其他規則的邊界表 | PC-166 列更新：標註規則 5 二相性對應 PC-166 防護 D 延伸（世界有而記錄無的反向漂移）|

### 設計決策

- **不引入 M1-M5 標籤**：規則自包含，二相性用描述性語言（記錄有而世界無 / 世界有而記錄無），不依賴 W1-039 taxonomy
- **跨專案 sync 安全**：規則本體無專案 ticket ID；footer 功能性描述
- **與規則 3 分工**：規則 3 是「固定值查證手段」，規則 5 是「為何必須查世界平面的原則」

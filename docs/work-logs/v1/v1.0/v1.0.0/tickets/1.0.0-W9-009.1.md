<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->
### 修法（W9-009 推薦方向 4：混合語意）

onboard_checker.py:
- 新增 _is_path_ignored_by_git(project_root, relative_path)：subprocess git check-ignore -q，returncode 0→True / 1→False / 128 非 repo 或 FileNotFoundError 無 git→None
- check_gitignore_completeness 新增內部 _rule_satisfied(probe, *patterns)：git 判定代表路徑，None 時 fallback any(_has_gitignore_rule)
- 7 個 has_X 改用 _rule_satisfied，代表路徑：coverage/_probe、htmlcov/_probe、.claude/hook-logs/_probe、worktrees/_probe、tool-results/_probe、handoff/_probe、.claude/dispatch-active.json（實際檔名）、__pycache__/_probe
- _has_gitignore_rule 保留為 fallback 層不刪（無 git repo 時行為不變）

### 測試
- test_gitignore_git_semantic_recognizes_legal_variants：git repo + **/hook-logs/ + 具體 .json，RED→GREEN

- test_gitignore_git_semantic_covers_all_variant_syntaxes：4 種語法（**/、/錨定、.*、具體檔名）全覆蓋

### 本專案修補
.gitignore 補 .claude/tool-results/（W9-009 重現實驗確認真缺）

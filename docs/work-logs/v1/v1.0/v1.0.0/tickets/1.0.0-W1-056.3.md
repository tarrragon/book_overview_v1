### 問題根因

### 影響範圍

- `.claude/settings.json`（設定檔）
- 每 session system-reminder 的 skills 清單注入量

### 相關 Error Pattern

無直接相關 error pattern。與 W16-003（skills 注入成本分析）為延續關係。

### 前置發現

W16-003（2026-04-19）已完成全量分析，W16-003.1 已卸載 document-skills + example-skills。但 `claude plugin list` 顯示兩者**目前已重新安裝**（可能用戶手動或 CC 更新自動恢復）。`disableBundledSkills` 是 CC 2.1.169 新增的 settings.json 層級控制，比 plugin uninstall 更持久且不影響其他專案。

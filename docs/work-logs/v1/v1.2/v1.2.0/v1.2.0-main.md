# v1.2.0 版本工作日誌

**版本號**: v1.2.0
**開始日期**: 2026-05-12
**完成日期**: 2026-06-22
**目標**: QR 離線同步 + JSON 匯入 + Popup UI 重構 + Chrome DevTools MCP attach 模式驗證
**狀態**: 已完成

---

## 前情提要

v1.2.0 承接 v1.1.0（監測任務集中版本）後，聚焦於 Extension 端核心功能開發：

1. **QR 離線同步**（W1-001, W1-006）：實作 QR 動畫產生模組，支援 gzip 壓縮 + 分塊 + Canvas 輪播
2. **JSON 匯入**（W1-002, W1-005, W1-007, W1-008）：Popup JSON 匯入 UI，含防舊蓋新 + 智慧合併
3. **跨端整合測試**（W2-001）：Extension QR 動畫 + App JSON 匯出入端到端驗證
4. **Popup UI 重構**（W2-002, W2-003, W2-005, W2-006）：常數集中管理 + 元件工廠模組化
5. **Chrome DevTools MCP attach 驗證**（W1-004）：Chrome 149 釋出後實測 --browserUrl + --categoryExtensions 可用

## 進度追蹤

## 技術筆記

### Chrome DevTools MCP attach 模式（W1-004）

Chrome 149.0.7827.155 stable 釋出（2026-06-02）後，chrome-devtools-mcp v1.3.0 的 `--browserUrl` + `--categoryExtensions` 組合已可正常使用。5 個 extension 工具全部暴露。`.mcp.json` 已從 `--userDataDir`（自啟動）切換為 `--browserUrl=http://127.0.0.1:9222`（attach 模式）。

### 權限變更

本版本無 manifest.json permissions / host_permissions 變更。manifest.json 版本從 1.1.0 同步至 1.2.0。

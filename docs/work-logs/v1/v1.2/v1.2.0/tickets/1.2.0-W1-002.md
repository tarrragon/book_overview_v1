**命名規範**：ticket 規劃 `src/sync/json-importer.js`，建議改為 `src/import/json-importer.js`（匯入是單向操作，sync 語意過重；且與 `src/export/` 形成對等目錄）。函式命名（`importBooks` / `validateSyncMeta`）符合既有 camelCase 慣例。事件命名應對齊 `src/export/export-events.js` 的模式。

### 3. API 介面定義

#### 3.1 parseAndValidate(fileContent: string) -> ParseResult | ImportError

解析 JSON 字串並驗證結構。

輸入：
- fileContent: string — File API 讀取的原始文字內容（必填）

輸出（成功）：
  ParseResult {
    data: Object,          // 解析後的 JSON 物件
    source: string,        // 'canonical' | 'v2' | 'app-legacy' | 'v1'
    metadata: {
      exportedAt: string | null,  // ISO8601 時戳（canonical 格式）
      sourceApp: string | null,
      totalBooks: number | null
    },
    bookCount: number      // 實際 books 陣列長度
  }

輸出（失敗）：
  ImportError { code: string, message: string }

處理流程：
1. JSON.parse(fileContent) — 失敗回傳 IMPORT_PARSE_ERROR
2. detectInterchangeSource(data) — 回傳 null 則 IMPORT_UNKNOWN_FORMAT
3. 提取 metadata（canonical 讀 data.metadata，其餘格式為 null）
4. 計算 bookCount
5. totalBooks 與 bookCount 不一致時記錄 warning（不阻擋匯入）

#### 3.2 checkStaleness(exportedAt: string | null) -> StalenessResult

比較匯出時間與本機上次匯入時間，防舊蓋新。

輸入：exportedAt: string | null — ISO8601 匯出時間

輸出：
  StalenessResult {
    isStale: boolean,
    exportedAt: Date | null,
    lastImportedAt: Date | null
  }

處理流程：
1. 從 chrome.storage.local 讀取 last_imported_at key
2. exportedAt 為 null → isStale = false（無法判斷時不阻擋）
3. lastImportedAt 為 null → isStale = false（首次匯入）
4. new Date(exportedAt) < new Date(lastImportedAt) → isStale = true

#### 3.3 executeImport(fileContent: string, options?: ImportOptions) -> ImportResult

完整匯入流程編排。

輸入：
- fileContent: string — File API 讀取的原始文字內容（必填）
- options?: { skipStalenessCheck: boolean }（預設 false）

輸出（成功）：
  ImportResult {
    success: true,
    summary: { added: number, updated: number, unchanged: number, total: number },
    source: string,
    importedAt: string
  }

輸出（失敗）：
  ImportResult { success: false, error: ImportError }

處理流程：
1. 呼叫 parseAndValidate(fileContent)
2. 若失敗 → 回傳錯誤
3. 呼叫 checkStaleness(metadata.exportedAt)
4. 若 isStale && !skipStalenessCheck → 回傳 IMPORT_STALE_DATA（含 staleness 資訊）
5. 依 source 分流轉換：
   - canonical → data.books.map(mapCanonicalToV1Book)；tagTree 直接傳遞
   - app-legacy → convertAppLegacyToV2Data(data)
   - v2 / v1 → books 直接使用
6. 呼叫 mergeAllData({ books, tags, tagCategories })
7. 若 mergeAllData 回傳 success: false → 回傳 IMPORT_STORAGE_ERROR
8. 計算合併摘要（added / updated / unchanged）
9. 將 importedAt 寫入 chrome.storage.local 的 last_imported_at key
10. 回傳成功結果

合併摘要計算：需在 mergeAllData 呼叫前後比對本機 books 數量與 id 集合。
  added = 合併後新出現的 id 數量
  updated = 合併前後都存在但內容變更的 id 數量
  unchanged = total - added - updated

### 4. 資料結構

#### 4.1 last_imported_at Storage Key

Storage API: chrome.storage.local
Key 名稱: last_imported_at
值格式: ISO8601 字串
初始值: 不存在（首次匯入前）
更新時機: executeImport 成功完成後

SA A4 建議整合至 sync-metadata-manager.js，但評估後 sync-metadata-manager.js 管理 chrome.storage.sync（跨裝置），last_imported_at 是本機行為記錄，應存 chrome.storage.local。獨立 key 即可。

#### 4.2 Metadata 讀取映射

| 來源格式 | exportedAt 來源 | sourceApp 來源 | totalBooks 來源 |
|---------|-----------------|----------------|-----------------|
| canonical | data.metadata.exportedAt | data.metadata.sourceApp | data.metadata.totalBooks |
| v2 | null | null | null |
| v1 | null | null | null |
| app-legacy | null | null | null |

### 6. 錯誤處理

| 錯誤碼 | 觸發條件 | 使用者訊息 |
|--------|---------|----------|
| IMPORT_PARSE_ERROR | JSON.parse 失敗 | 檔案格式錯誤，無法解析 JSON |
| IMPORT_UNKNOWN_FORMAT | detectInterchangeSource 回傳 null | 無法識別的檔案格式 |
| IMPORT_STALE_DATA | exportedAt < lastImportedAt | 匯入檔案較舊（非終止，可確認繼續） |
| IMPORT_STORAGE_ERROR | mergeAllData 回傳 success: false | 儲存失敗，請重試 |
| IMPORT_FILE_READ_ERROR | FileReader 讀取失敗 | 檔案讀取失敗 |
| IMPORT_EMPTY_BOOKS | books 陣列為空 | 匯入檔案中沒有書籍資料 |

### 9. Decision Questions

Q1（跨模組共用）：涉及 import + export + storage 三模組。detector/adapter 為純消費（只讀 API），不新增共用邏輯。結論：無新共用需求。

Q2：不適用。

Q3（ID 設計）：不新增 ID 欄位。匯入書籍 id 來自匯出檔案，mergeAllData 以 id 為主鍵做 LWW。結論：無碰撞風險。

Q4：不適用。

Q5（錯誤可觀測性）：所有 catch 區塊記錄 warning/error 日誌（專案 Logger）。使用者側以 UI 顯示錯誤訊息（複用 errorContainer）。靜默場景：totalBooks 不一致僅記 warning。

Q6a（最慢操作）：mergeAllData 涉及 storage 三讀三寫。1000 本書預期 < 500ms。可接受延遲上限：3 秒。

Q6b（寫入失敗）：依賴 mergeAllData 既有 withAtomicRollback。策略：回滾（已實作）。

#### 4. 虛擬碼

##### 4.1 parseAndValidate

```
FUNCTION parseAndValidate(fileContent):
    IF fileContent is null/undefined OR not string:
        RETURN ImportError(IMPORT_PARSE_ERROR)

    TRY: parsedData = JSON.parse(fileContent)
    CATCH: LOG warn; RETURN ImportError(IMPORT_PARSE_ERROR)

    source = detectInterchangeSource(parsedData)
    IF source is null: LOG warn; RETURN ImportError(IMPORT_UNKNOWN_FORMAT)

    metadata = { exportedAt:null, sourceApp:null, totalBooks:null }
    IF source==='canonical' AND parsedData.metadata:
        metadata = pick(parsedData.metadata, [exportedAt, sourceApp, totalBooks])

    books = extractBooksArray(parsedData, source)
    IF books.length === 0: RETURN ImportError(IMPORT_EMPTY_BOOKS)

    IF metadata.totalBooks !== null AND metadata.totalBooks !== books.length:
        LOG warn("totalBooks mismatch")

    RETURN { data:parsedData, source, metadata, bookCount:books.length }

FUNCTION extractBooksArray(data, source):
    IF source==='v1' AND Array.isArray(data): RETURN data
    IF data.books AND Array.isArray(data.books): RETURN data.books
    RETURN []
```

##### 4.2 checkStaleness

```
ASYNC FUNCTION checkStaleness(exportedAt):
    stored = AWAIT chrome.storage.local.get(['last_imported_at'])
    lastImportedAt = stored.last_imported_at OR null

    IF exportedAt is null: RETURN { isStale:false, exportedAt:null, lastImportedAt }
    IF lastImportedAt is null: RETURN { isStale:false, exportedAt, lastImportedAt:null }

    TRY:
        exp = new Date(exportedAt); imp = new Date(lastImportedAt)
        IF isNaN(exp) OR isNaN(imp): LOG warn; RETURN { isStale:false, ... }
        isStale = exp < imp
    CATCH: LOG warn; isStale = false

    RETURN { isStale, exportedAt, lastImportedAt }
```

##### 4.3 executeImport

```
ASYNC FUNCTION executeImport(fileContent, options={}):
    parseResult = parseAndValidate(fileContent)
    IF parseResult is error: RETURN { success:false, error:parseResult }

    staleness = AWAIT checkStaleness(parseResult.metadata.exportedAt)
    IF staleness.isStale AND NOT options.skipStalenessCheck:
        RETURN { success:false, error:IMPORT_STALE_DATA, staleness }

    // 來源分流
    { books, tags, tagCategories } = convertBySource(parseResult)

    // 快照 + 合併 + 摘要
    beforeMap = AWAIT snapshotLocalBooks()
    mergeResult = AWAIT mergeAllData({ books, tags, tagCategories })
    IF NOT mergeResult.success: LOG error; RETURN { success:false, error:IMPORT_STORAGE_ERROR }

    afterBooks = AWAIT loadLocalBooks()
    summary = computeSummary(beforeMap, afterBooks)

    importedAt = now().toISOString()
    AWAIT chrome.storage.local.set({ last_imported_at: importedAt })

    RETURN { success:true, summary, source:parseResult.source, importedAt }
```

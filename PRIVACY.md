# 隱私權政策

**最後更新日期**：2026-05-23
**適用對象**：Readmoo 書庫管理器 Chrome 擴充功能

## 概述

Readmoo 書庫管理器（以下稱「本擴充功能」）尊重並保護使用者隱私。本擴充功能將所有資料保存於使用者本機裝置，不蒐集、不傳輸、不販售任何個人資料。

## 蒐集的資料

本擴充功能在使用者主動觸發提取時，從 Readmoo 書庫頁面讀取下列書目資料：

- 書名、作者、出版社
- 封面圖片網址
- 閱讀進度與閱讀狀態
- 書籍類型
- 使用者自訂標籤

本擴充功能**不**蒐集帳號、密碼、Email、付款資訊、瀏覽紀錄，或任何其他個人識別資訊（PII）。

## 資料的使用方式

蒐集的書目資料僅用於本擴充功能的核心功能：本機書庫管理、搜尋、篩選、統計與匯出。資料不會用於上述以外的任何用途。

## 資料儲存與傳輸

- **儲存位置**：所有資料儲存於使用者瀏覽器本機（`chrome.storage.local`）。
- **資料傳輸**：本擴充功能完全離線運作，不將任何資料傳輸至開發者伺服器或任何第三方。
- **資料共享**：本擴充功能不與任何第三方共享、不販售、不轉讓使用者資料。

## 第三方服務

本擴充功能不整合任何第三方分析（analytics）、廣告或追蹤服務，亦不載入任何遠端程式碼。

## 使用者控制權

- **匯出**：使用者可隨時將書庫資料匯出為 JSON 或 CSV 檔案，檔案下載至使用者本機。
- **刪除**：使用者可透過瀏覽器的擴充功能管理頁面移除本擴充功能；移除後 `chrome.storage.local` 中的相關資料將一併清除。

## 權限說明

本擴充功能採最小權限原則。完整的權限宣告與使用理由詳見專案 README 的「權限與隱私聲明」章節，摘要如下：

| 權限 | 用途 |
|------|------|
| `storage` | 於使用者本機儲存提取的書庫資料 |
| `activeTab` | 於使用者點擊擴充功能圖示時，存取當前 Readmoo 分頁以觸發提取 |
| `*://*.readmoo.com/*` | 允許在 Readmoo 網域讀取書庫頁面內容以擷取書目資料 |

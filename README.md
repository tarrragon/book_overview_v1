# Readmoo 書庫數據提取器 Chrome Extension

> 自動提取 Readmoo 電子書書庫資料，並提供本地瀏覽與匯出功能的 Chrome 擴充功能

## 🚀 快速開始

```bash
# 安裝依賴
npm install --legacy-peer-deps

# 執行測試
npm test

# 建置開發版本
npm run build:dev

# 建置生產版本
npm run build:prod
```

## 🚀 3 步驟開始使用

### 步驟 1: 安裝擴充功能

1. 前往 [Chrome Web Store](https://chrome.google.com/webstore) 搜尋「Readmoo 書庫管理器」
2. 點擊「加到 Chrome」安裝
3. 確認擴充功能圖示出現在瀏覽器工具列

### 步驟 2: 登入並提取（以 Readmoo 為例）

1. 前往 [Readmoo 官網](https://readmoo.com) 並登入帳戶
2. 進入「我的書櫃」頁面
3. 點擊擴充功能圖示，按下「開始提取書籍資料」

> 💡 **未來版本**: 將支援 Kobo、Kindle、BookWalker 等多個書城平台

### 步驟 3: 管理您的書庫

1. 提取完成後點擊「查看書庫總覽」
2. 使用搜尋、篩選功能整理書籍
3. 需要時可匯出 CSV 檔案備份資料

> 💡 **提示**: 首次使用建議先提取少量書籍測試功能

## 🤔 常見問題

### Q: 會不會影響書城網站運行？

**A:** 不會。本擴充功能採用被動式資料讀取，不修改網站內容，對原網站無任何影響。目前支援 Readmoo，未來將以相同安全方式支援其他書城。

### Q: 提取的資料安全嗎？

**A:** 完全安全。所有資料僅存在您的電腦中，絕不上傳到任何伺服器，充分保護您的隱私。

### Q: 支援多少本書籍？

**A:** 理論上無限制。已測試支援 500+ 本書籍的使用者，運行流暢穩定。

### Q: 如果書城平台改版會受影響嗎？

**A:** 我們會密切關注各書城的更新，並及時推出相容性更新，確保功能正常運作。目前 v1.0 專注維護 Readmoo 相容性。

## 權限與隱私聲明

### 單一用途說明

本擴充功能的單一用途為：在使用者主動操作下，從 Readmoo 電子書平台提取個人書庫的書目資料，並於本機提供管理、搜尋、篩選與匯出功能。

### 權限使用理由

本擴充功能採最小權限原則，僅宣告以下三項權限：

| 權限 | 類型 | 使用理由 |
|------|------|---------|
| `storage` | API 權限 | 將提取的書庫資料儲存於瀏覽器本機（`chrome.storage.local`），供後續管理、搜尋與匯出使用。不同步至雲端、不用於追蹤。 |
| `activeTab` | API 權限 | 僅在使用者點擊擴充功能圖示時，存取當前作用中的 Readmoo 分頁以觸發提取。不在背景監控或存取其他分頁。 |
| `*://*.readmoo.com/*` | 主機權限（host permission） | 允許 content script 在 Readmoo 網域注入並讀取書庫頁面內容以擷取書目資料。範圍限定 Readmoo 網域，不存取其他任何網站。 |

本擴充功能**不**使用下列常受審核關注的高風險權限：`tabs`（廣域分頁存取）、`<all_urls>`（全網域）、`webRequest`（攔截網路請求），亦不載入任何遠端程式碼，完全符合 Manifest V3 規範。

### 資料用途聲明

| 項目 | 說明 |
|------|------|
| 蒐集的資料 | 書目資料：書名、作者、封面圖網址、閱讀進度、閱讀狀態、書籍類型、自訂標籤。 |
| 儲存位置 | 僅儲存於使用者本機（`chrome.storage.local`），不上傳任何伺服器。 |
| 資料傳輸 | 不傳輸任何資料至開發者或第三方。擴充功能完全離線運作。 |
| 個人識別資訊 | 不蒐集帳號、密碼、Email、付款資訊等個人識別資訊（PII）。 |
| 分析與廣告 | 不含使用行為分析（analytics）、不含廣告、不含任何回報機制。 |
| 匯出檔案 | JSON / CSV 匯出檔由使用者主動下載至本機，內容與去向完全由使用者掌控。 |

本擴充功能不出售、不轉讓使用者資料，資料使用範圍不超出上述單一用途，符合 Chrome Web Store 開發者計畫政策。

### 隱私權政策

完整隱私權政策見專案根目錄的 [PRIVACY.md](./PRIVACY.md)。摘要：本擴充功能將所有書庫資料保存於使用者裝置本機，不蒐集個人識別資訊、不進行任何網路傳輸，亦不與任何第三方共享資料。
